# VP-049: sbctl Unified CLI with OpenSSH Auth

## Property Statement

sbctl with a valid --key, targeting each daemon type:
  router → successful response
  ∧ access → successful response
  ∧ console → successful response
  ∧ control → successful response

A single sbctl binary must authenticate successfully against all four daemon
types using the same OpenSSH-style key mechanism. No daemon-specific client
binary is required.

## Source Contract

- **BC:** BC-2.07.002 — sbctl Unified CLI Authenticates Against All Daemon Types
- **Postcondition/Invariant:** Postcondition — "sbctl with valid key returns a valid response from router, access, console, and control daemons."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| e2e | go test (testing.T) | yes | full four-daemon stack; sbctl status against each; assert exit 0 and non-empty response |

## Proof Harness Skeleton

```go
//go:build integration

package sbctl_test

import (
	"bytes"
	"context"
	"os/exec"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/testenv"
)

// TestE2E_Sbctl_UnifiedCLI_OpenSSHAuth verifies that sbctl authenticates
// successfully against all four daemon types with a single key.
func TestE2E_Sbctl_UnifiedCLI_OpenSSHAuth(t *testing.T) {
	t.Parallel()

	ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
	t.Cleanup(cancel)

	env := testenv.NewFull(t, ctx) // router + access + console + control
	t.Cleanup(env.Close)

	keyFile := env.WriteKeyFile(t)

	daemons := []struct {
		name   string
		target string
	}{
		{"router", env.RouterAddr(t)},
		{"access", env.AccessAddr(t)},
		{"console", env.ConsoleAddr(t)},
		{"control", env.ControlAddr(t)},
	}

	for _, d := range daemons {
		d := d
		t.Run(d.name, func(t *testing.T) {
			t.Parallel()

			cmd := exec.CommandContext(ctx, "sbctl",
				"--key", keyFile,
				"--target", d.target,
				"status",
			)
			var stdout, stderr bytes.Buffer
			cmd.Stdout = &stdout
			cmd.Stderr = &stderr

			if err := cmd.Run(); err != nil {
				t.Errorf("sbctl status against %s failed: %v\nstderr: %s",
					d.name, err, stderr.String())
				return
			}
			if len(bytes.TrimSpace(stdout.Bytes())) == 0 {
				t.Errorf("sbctl status against %s returned empty response", d.name)
			}
		})
	}

	// Allow subtests to complete within the outer timeout.
	time.Sleep(100 * time.Millisecond)
}
```
