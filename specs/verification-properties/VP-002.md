# VP-002: ParseOuterHeader Rejects Version Mismatch

## Property Statement

∀ byte slice b where len(b) >= 44 ∧ (b[0] >> 4) != 0:
  ParseOuterHeader(b) returns ErrVersionMismatch

The version field occupies byte 0 only (1 byte). It is a packed field:
  bits [7:4] = major version (0–15)
  bits [3:0] = minor version (0–15)

For protocol v0.1, the version byte is 0x01 (major=0, minor=1).
For protocol v1.0, the version byte is 0x10 (major=1, minor=0).

The parser checks the major nibble (b[0] >> 4). Current major version is 0.
Any encoding where the major nibble does not equal 0 causes ParseOuterHeader
to return ErrVersionMismatch with no partially-decoded header.

This property covers 15 non-zero major nibble values (0x1–0xF in the high nibble).

## Source Contract

- **BC:** BC-2.01.004 — Frame outer-header encoding and decoding at 44-byte fixed layout
- **Postcondition/Invariant:** Edge Case EC-001 — "Frame with version field indicating major version > current: Router rejects frame with E-PRT-001; does not forward."

## Proof Harness Skeleton

```go
package frame_test

import (
	"errors"
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/frame"
)

// genBufWithBadMajorNibble produces a 44-byte buffer where the version byte
// (offset 0) has a non-zero major nibble (bits[7:4] != 0).
//
// Version encoding (ARCH-02): byte 0, bits[7:4]=major, bits[3:0]=minor.
// Current major version is 0. Any major nibble 0x1–0xF is invalid.
func genBufWithBadMajorNibble() gopter.Gen {
	// Major nibble values != 0: 1–15 (shifted to high nibble: 0x10, 0x20, ..., 0xF0)
	// Combined with arbitrary minor nibble: (major << 4) | minor
	return gopter.CombineGens(
		gen.IntRange(1, 15),           // major nibble: 1–15
		gen.IntRange(0, 15),           // minor nibble: 0–15
		gen.SliceOfN(43, gen.UInt8()), // remainder of buffer
	).Map(func(vals []interface{}) []byte {
		major := byte(vals[0].(int))
		minor := byte(vals[1].(int))
		rest := vals[2].([]byte)
		buf := make([]byte, 44)
		buf[0] = (major << 4) | minor
		copy(buf[1:], rest)
		return buf
	})
}

func TestPropVersionMismatch_ParseOuterHeader(t *testing.T) {
	properties := gopter.NewProperties(nil)

	properties.Property("parser returns ErrVersionMismatch for major nibble != 0 (i.e. nibble 0x1–0xF)", prop.ForAll(
		func(buf []byte) bool {
			_, err := frame.ParseOuterHeader(buf)
			return errors.Is(err, frame.ErrVersionMismatch)
		},
		genBufWithBadMajorNibble(),
	))

	properties.TestingRun(t)
}
```
