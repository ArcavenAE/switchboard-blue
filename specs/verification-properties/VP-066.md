# VP-066: Management Server Enforces Bounded Read — Message > 64 KiB Causes Error and Connection Close, No OOM (CWE-400)

## Property Statement

∀ connection to `mgmt.Server` where the peer sends a single message (any
message in the handshake or RPC phase) whose raw byte count exceeds
`mgmt.MaxMessageBytes` (64 KiB = 65536 bytes):

1. The `json.Decoder.Decode` call wrapping `io.LimitReader(conn, mgmt.MaxMessageBytes)`
   returns a non-nil error.
2. The server closes the connection.
3. The server's heap allocation for this connection does not grow unboundedly —
   no more than `MaxMessageBytes` (64 KiB) of data is read from the connection
   for the oversized message.
4. The server process does not OOM, panic, or become unresponsive to subsequent
   connections.

This property applies symmetrically to all read phases: the initial CHALLENGE_RESPONSE
message, any subsequent RPC request message, and any other message. There is no
code path in `internal/mgmt` that reads from a connection without `io.LimitReader`.

The constant `mgmt.MaxMessageBytes = 1 << 16` (64 KiB) is the enforcement boundary
per ADR-012 §6 (Bounded Read, CWE-400).

## Source Contract

- **BC:** BC-2.07.004 — Daemon Management Server Authenticates All Connections via
  Ed25519 Challenge-Response (Fail-Closed)
- **Postcondition PC-6:** every `json.Decoder.Decode` preceded by
  `io.LimitReader(conn, MaxMessageBytes)`; oversized message → error + close; no OOM.
- **Invariant 3:** all socket reads are bounded by MaxMessageBytes; no unbounded-read
  path in `internal/mgmt`.

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| unit | `go test` | yes | Exact-boundary (65535 bytes OK, 65537 bytes triggers limit) |
| fuzz | `go test -fuzz` | no | Arbitrary byte sequences up to 2×MaxMessageBytes; assert no OOM/panic |

## Proof Harness Skeleton

```go
package mgmt_test

import (
    "bytes"
    "context"
    "crypto/ed25519"
    "crypto/rand"
    "io"
    "net"
    "path/filepath"
    "testing"
    "time"

    "github.com/arcavenae/switchboard/internal/mgmt"
)

// TestMgmtServer_BoundedRead_VP066 verifies VP-066 (unit component):
// a message exceeding MaxMessageBytes causes the connection to close.
func TestMgmtServer_BoundedRead_VP066(t *testing.T) {
    t.Parallel()

    _, daemonPriv, err := ed25519.GenerateKey(rand.Reader)
    if err != nil {
        t.Fatalf("generate daemon key: %v", err)
    }
    keySet := mgmt.NewOperatorKeySet(nil) // empty: bootstrap mode (daemon key only)

    t.Run("oversized_message_closes_connection", func(t *testing.T) {
        t.Parallel()
        sockDir := t.TempDir()
        sockPath := filepath.Join(sockDir, "mgmt.sock")
        ln, err := net.Listen("unix", sockPath)
        if err != nil {
            t.Fatalf("listen: %v", err)
        }
        defer ln.Close()
        go mgmt.NewServer(ln, daemonPriv, keySet, nil, "dev").Serve(context.Background()) //nolint:errcheck
        client, err := net.Dial("unix", sockPath)
        if err != nil {
            t.Fatalf("dial: %v", err)
        }
        defer client.Close()

        // Read CHALLENGE (server-first, so we must consume it first).
        buf := make([]byte, mgmt.MaxMessageBytes)
        n, err := client.Read(buf)
        if err != nil {
            t.Fatalf("read challenge: %v", err)
        }
        _ = buf[:n] // discard challenge

        // Send a message larger than MaxMessageBytes.
        oversized := bytes.Repeat([]byte("x"), mgmt.MaxMessageBytes+1)
        // Wrap in a JSON-ish envelope so decoder starts parsing.
        payload := append([]byte(`{"type":"challenge_response","nonce_sig":"`), oversized...)
        payload = append(payload, '"', '}', '\n')

        _, writeErr := client.Write(payload)
        if writeErr != nil {
            // Connection may already be closed — that's acceptable.
            return
        }

        // Server should close the connection shortly after the read limit is hit.
        client.SetReadDeadline(time.Now().Add(500 * time.Millisecond))
        readBuf := make([]byte, 1024)
        _, readErr := client.Read(readBuf)
        if readErr == nil {
            t.Error("expected connection close after oversized message; got successful read")
        }
    })
}

// FuzzMgmtServer_BoundedRead_VP066 verifies VP-066 (fuzz component):
// arbitrary byte sequences up to 2×MaxMessageBytes never cause OOM or panic.
func FuzzMgmtServer_BoundedRead_VP066(f *testing.F) {
    f.Add([]byte(`{"type":"challenge_response","nonce_sig":"AAAA","pubkey":"BBBB"}`))
    f.Add(bytes.Repeat([]byte("A"), mgmt.MaxMessageBytes+512))
    f.Add([]byte("not json at all"))
    f.Add([]byte{0x00, 0xFF, 0xFE})

    _, daemonPriv, _ := ed25519.GenerateKey(rand.Reader)
    keySet := mgmt.NewOperatorKeySet(nil)

    f.Fuzz(func(t *testing.T, data []byte) {
        if len(data) > 2*mgmt.MaxMessageBytes {
            data = data[:2*mgmt.MaxMessageBytes]
        }

        sockDir := t.TempDir()
        sockPath := filepath.Join(sockDir, "mgmt.sock")
        ln, err := net.Listen("unix", sockPath)
        if err != nil {
            t.Fatalf("listen: %v", err)
        }
        defer ln.Close()
        go mgmt.NewServer(ln, daemonPriv, keySet, nil, "dev").Serve(context.Background()) //nolint:errcheck
        client, err := net.Dial("unix", sockPath)
        if err != nil {
            t.Fatalf("dial: %v", err)
        }
        defer client.Close()

        // Consume the server-sent CHALLENGE.
        client.SetReadDeadline(time.Now().Add(100 * time.Millisecond))
        io.Copy(io.Discard, io.LimitReader(client, int64(mgmt.MaxMessageBytes))) //nolint:errcheck

        // Send the fuzz payload.
        client.SetWriteDeadline(time.Now().Add(100 * time.Millisecond))
        client.Write(data) //nolint:errcheck

        // The property: server must not panic and must close the connection
        // within a reasonable deadline. OOM would manifest as test timeout.
        client.SetReadDeadline(time.Now().Add(200 * time.Millisecond))
        io.Copy(io.Discard, client) //nolint:errcheck
        // If we reach here without panic/timeout, the property holds.
    })
}
```
