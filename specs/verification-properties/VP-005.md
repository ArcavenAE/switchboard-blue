# VP-005: VerifyHMAC Rejects Single-Bit Tag Flips

## Property Statement

∀ key []byte, msg []byte, tag [8]byte where tag == ComputeHMAC(key, msg):
  ∀ i ∈ [0, 63]:  // 64 bits in 8-byte tag
    VerifyHMAC(key, msg, flipBit(tag, i)) == false

Flipping any single bit in a valid HMAC tag must cause VerifyHMAC to return
false. This property is the minimum security bar for truncated HMAC-SHA256:
no single-bit corruption or targeted forgery attempt on the tag field passes
verification.

The `tag` parameter is `[8]byte` (array type, not slice). Array assignment in
Go copies all 8 bytes — there is no shared backing array between the original
tag and the flipped copy. This eliminates the aliasing bug where a slice
assignment (`flipped := tag` with `tag []byte`) would share the backing array
and cause all 64 bit-flip iterations to mutate the same bytes, turning the
property into a no-op.

Canonical function signatures:
```go
func ComputeHMAC(key []byte, msg []byte) [8]byte
func VerifyHMAC(key []byte, msg []byte, tag [8]byte) bool
func hmacTagIsBinding(key [32]byte, msg []byte, tag [8]byte) bool
```

## Proof Harness Skeleton

```go
package hmac_test

import (
	"testing"

	"github.com/arcavenae/switchboard/internal/hmac"
)

// FuzzVerifyHMAC_RejectsBitFlips verifies that a single-bit corruption of any
// tag byte causes VerifyHMAC to return false.
func FuzzVerifyHMAC_RejectsBitFlips(f *testing.F) {
	// Seed corpus: representative key and frame sizes.
	f.Add(
		[]byte("test-admission-key-32-bytes-long!"),
		[]byte("frame payload for fuzz testing"),
	)
	f.Add(
		[]byte{0xFF, 0x00, 0xAB, 0xCD},
		[]byte{},
	)

	f.Fuzz(func(t *testing.T, key, frameBytes []byte) {
		if len(key) == 0 {
			t.Skip()
		}
		tag := hmac.ComputeHMAC(key, frameBytes) // tag is [8]byte (array, not slice)

		// Exercise every single-bit flip across the 8-byte tag.
		// tag is [8]byte (array type). Assignment copies the array — no aliasing.
		// Each iteration starts from the original tag, flips one bit, and checks.
		for byteIdx := 0; byteIdx < 8; byteIdx++ {
			for bitIdx := 0; bitIdx < 8; bitIdx++ {
				flipped := tag                         // array copy — no shared backing
				flipped[byteIdx] ^= 1 << uint(bitIdx) // mutates the copy only
				if hmac.VerifyHMAC(key, frameBytes, flipped) {
					t.Errorf(
						"VerifyHMAC accepted flipped tag: byte=%d bit=%d key=%x frame=%x",
						byteIdx, bitIdx, key, frameBytes,
					)
				}
			}
		}
	})
}
```
