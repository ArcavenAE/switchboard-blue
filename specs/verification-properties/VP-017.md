# VP-017: HalfChannel Sequence Increments by Exactly 1

## Property Statement

∀ consecutive Tick calls on the same HalfChannel:
  let f1 = Tick(state, payload1)
  let f2 = Tick(state', payload2)
  f2.Sequence == f1.Sequence + 1

The sequence counter must increment by exactly one per tick — never zero,
never more than one, never skipping values. This invariant enables the
receiver to detect dropped frames unambiguously.

## Source Contract

- **BC:** BC-2.01.003 — HalfChannel Sequence Counter Increments on Every Tick
- **Postcondition/Invariant:** Invariant — "Sequence is strictly monotone with step 1."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| proptest | gopter v0.2.9+ | no | random starting sequence and N consecutive ticks; asserts sequence[i+1] == sequence[i] + 1 for all i |

## Proof Harness Skeleton

```go
package halfchannel_test

import (
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/halfchannel"
)

func TestProp_Tick_SequenceIncrementsByOne(t *testing.T) {
	properties := gopter.NewProperties(nil)

	properties.Property("sequence increments by exactly 1 on each Tick", prop.ForAll(
		func(startSeq uint64, n uint8, payload []byte) bool {
			if n == 0 {
				n = 1
			}
			ch := halfchannel.New(halfchannel.State{Sequence: startSeq})

			prev := startSeq - 1 // underflow intentional; first check uses startSeq
			isFirst := true
			for i := 0; i < int(n); i++ {
				frames := ch.Tick(payload)
				if len(frames) != 1 {
					return false
				}
				seq := frames[0].Sequence
				if isFirst {
					if seq != startSeq {
						return false
					}
					isFirst = false
				} else {
					if seq != prev+1 {
						return false
					}
				}
				prev = seq
			}
			return true
		},
		gen.UInt64(),
		gen.UInt8(),
		gen.SliceOf(gen.UInt8()),
	))

	properties.TestingRun(t)
}
```
