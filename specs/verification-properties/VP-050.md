# VP-050: Console Remotely Controllable via sbctl

## Property Statement

`sbctl console attach/detach/switch` commands sent to a running console daemon:
  are executed
  ∧ the console state changes accordingly

The sbctl CLI must be able to drive all three console lifecycle commands
remotely. Each command must produce an observable state change in the
console daemon.

## Source Contract

- **BC:** BC-2.08.001 — Console remotely controllable via sbctl: attach, detach, switch session, navigate
- **Postcondition/Invariant:** Postcondition — "attach/detach/switch commands execute and produce observable console state changes."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| e2e | go test (testing.T) | yes | running console daemon; sbctl attach → console attached; sbctl detach → console detached; sbctl switch → session changed |

## Proof Harness Skeleton

```go
//go:build integration

package sbctl_test

import (
	"bytes"
	"context"
	"os/exec"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/testenv"
)

// TestE2E_Sbctl_Console_RemoteControl verifies that sbctl console
// attach/detach/switch commands change the running console daemon's state.
func TestE2E_Sbctl_Console_RemoteControl(t *testing.T) {
	t.Parallel()

	ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
	t.Cleanup(cancel)

	env := testenv.NewFull(t, ctx)
	t.Cleanup(env.Close)

	sessionA := env.CreateSession(t)
	sessionB := env.CreateSession(t)
	consoleAddr := env.ConsoleAddr(t)
	keyFile := env.WriteKeyFile(t)

	sbctl := func(t *testing.T, args ...string) (stdout, stderr string) {
		t.Helper()
		base := []string{"--key", keyFile, "--target", consoleAddr}
		cmd := exec.CommandContext(ctx, "sbctl", append(base, args...)...)
		var out, errOut bytes.Buffer
		cmd.Stdout = &out
		cmd.Stderr = &errOut
		if err := cmd.Run(); err != nil {
			t.Fatalf("sbctl %v failed: %v\nstderr: %s", args, err, errOut.String())
		}
		return out.String(), errOut.String()
	}

	// --- attach ---
	sbctl(t, "console", "attach", "--session", sessionA)
	time.Sleep(200 * time.Millisecond)

	state := env.ConsoleState(t)
	if state.AttachedSession != sessionA {
		t.Errorf("after attach: expected session %s, got %s", sessionA, state.AttachedSession)
	}
	if !state.IsAttached {
		t.Error("after attach: console reports not attached")
	}

	// --- switch ---
	sbctl(t, "console", "switch", "--session", sessionB)
	time.Sleep(200 * time.Millisecond)

	state = env.ConsoleState(t)
	if state.AttachedSession != sessionB {
		t.Errorf("after switch: expected session %s, got %s", sessionB, state.AttachedSession)
	}

	// --- detach ---
	sbctl(t, "console", "detach")
	time.Sleep(200 * time.Millisecond)

	state = env.ConsoleState(t)
	if state.IsAttached {
		t.Error("after detach: console still reports attached")
	}
}
```
