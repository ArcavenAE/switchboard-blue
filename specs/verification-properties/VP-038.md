# VP-038: E→PE Graduation by Config Change Only

## Property Statement

An E router restarted with upstream_routers populated:
  enters PE mode
  ∧ does not require binary replacement
  ∧ does not require SVTN re-initialization

PE mode graduation must be achievable purely through configuration change
and restart — no code deployment, no SVTN teardown/rebuild.

## Source Contract

- **BC:** BC-2.09.001 — E router graduates to PE mode by adding upstream router connections in config
- **Postcondition/Invariant:** Postcondition — "Router enters PE mode after restart with upstream_routers set; no binary change required."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| e2e | go test (testing.T) | yes | start router in E mode; restart with upstream_routers config; assert mode is PE; assert SVTN still intact |

## Proof Harness Skeleton

```go
//go:build integration

package config_test

import (
	"context"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/testenv"
)

// TestE2E_EtoPE_GraduationByConfigChange verifies that adding
// upstream_routers to a router config and restarting promotes it to PE mode.
func TestE2E_EtoPE_GraduationByConfigChange(t *testing.T) {
	t.Parallel()

	ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
	t.Cleanup(cancel)

	// Start a PE (upstream) router that the E router will connect to.
	env := testenv.New(t, ctx)
	t.Cleanup(env.Close)

	peAddr := env.PERouterAddr(t)

	// Start the router in E mode (no upstream_routers).
	eRouter := env.StartRouter(t, testenv.RouterConfig{
		UpstreamRouters: nil,
	})
	if eRouter.Mode() != testenv.ModeE {
		t.Fatalf("expected E mode at startup, got %v", eRouter.Mode())
	}

	svtnIDBefore := eRouter.SVTNID()

	// Restart the same router binary with upstream_routers populated.
	eRouter.Restart(t, testenv.RouterConfig{
		UpstreamRouters: []string{peAddr},
	})

	// Allow startup.
	time.Sleep(500 * time.Millisecond)

	if eRouter.Mode() != testenv.ModePE {
		t.Errorf("expected PE mode after restart with upstream_routers, got %v", eRouter.Mode())
	}

	// SVTN must be unchanged.
	if eRouter.SVTNID() != svtnIDBefore {
		t.Errorf("SVTN ID changed after graduation: before=%v after=%v",
			svtnIDBefore, eRouter.SVTNID())
	}
}
```
