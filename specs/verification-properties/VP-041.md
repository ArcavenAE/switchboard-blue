# VP-041: Tick Regularity — p99 Jitter ≤ 2ms

## Property Statement

Over 1,000 consecutive ticks:
  p99(|actual_interval - configured_interval|) ≤ 2ms

This covers NFR-009: tick jitter must not exceed 2ms at the 99th percentile.
High jitter causes perceptible keystroke latency variation and degrades
the interactive experience even when mean latency is within bounds.

**Phase stratification:** The Phase-3 benchmark records `jitter_p99_ms` as an
observational metric only — no `b.Errorf` gate is asserted in Phase 3. Developer
hardware (e.g., M1 MacBook at 2.111ms p99) is not stable CI hardware. The ≤ 2ms
gate is enforced by VP-041 during Phase 6 formal verification on stable CI hardware.
See S-1.02 story revision 1.2 AC-005 patch and wave-1 consistency findings F-001 / F-002.

## Source Contract

- **BC:** BC-2.01.001 — Timeslice clock fires on every tick regardless of data availability
- **Postcondition/Invariant:** NFR — "p99 tick jitter ≤ 2ms over 1,000 consecutive ticks."

## Proof Harness Skeleton

```go
package halfchannel_test

import (
	"sort"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/halfchannel"
)

// BenchmarkTick_Jitter measures p99 deviation from the configured tick
// interval over 1,000 consecutive ticks.
//
// Phase-3: records jitter_p99_ms as observational metric only.
// Phase-6: adds the b.Errorf gate on stable CI hardware (AC-005, S-1.02 rev 1.2,
// wave-1 consistency F-001 resolution). Do NOT add a b.Errorf threshold check
// in Phase 3 or earlier.
func BenchmarkTick_Jitter(b *testing.B) {
	const (
		configuredInterval = 10 * time.Millisecond
		tickCount          = 1000
	)

	hc := halfchannel.New(1, halfchannel.Upstream, configuredInterval)
	deviations := make([]time.Duration, 0, tickCount)

	b.ResetTimer()
	prev := time.Now()
	for i := 0; i < tickCount; i++ {
		// Block until the next scheduled tick.
		time.Sleep(configuredInterval - time.Since(prev))
		now := time.Now()
		actual := now.Sub(prev)
		prev = now

		_ = hc.Tick()

		dev := actual - configuredInterval
		if dev < 0 {
			dev = -dev
		}
		deviations = append(deviations, dev)
	}
	b.StopTimer()

	sort.Slice(deviations, func(i, j int) bool {
		return deviations[i] < deviations[j]
	})

	p99idx := int(float64(len(deviations)) * 0.99)
	p99 := deviations[p99idx]

	// Phase-3: record metric only. Phase-6 adds the b.Errorf gate on stable
	// CI hardware (AC-005, S-1.02 story revision 1.2, wave-1 consistency F-001).
	b.ReportMetric(float64(p99)/float64(time.Millisecond), "jitter_p99_ms")
}
```
