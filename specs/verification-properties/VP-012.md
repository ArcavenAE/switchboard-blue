# VP-012: SessionAuth Rejects Unauthorized Console Key

## Property Statement

∀ auth_list AuthList, console_key PublicKey where console_key ∉ auth_list:
  Authorize(console_key, auth_list) == false

A console presenting a public key that is not on the session's per-session
authorization list must be denied, regardless of Tier 1 SVTN admission
status. Authorization is evaluated per-session by the access node;
the router is not involved. No key absent from auth_list may yield true.

## Source Contract

- **BC:** BC-2.05.003 — Per-session Tier 2 authorization enforced by access node, not router
- **Postcondition/Invariant:** Postcondition 2 — "If console's public key is NOT in the session authorization list: access node rejects the attach request with E-ADM-006." Invariant DI-010 — "Session authorization is access-node-enforced."

```go
package session_test

import (
	"errors"
	"fmt"
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/session"
)

// TestPropSessionAuth_RejectsUnauthorizedKey verifies VP-012: a console key that
// has never been registered via RegisterKey is always denied by Authorize,
// regardless of how many other keys are in the auth list.
//
// NOTE — illustrative skeleton only; execution deferred to Phase 6.
// The shipped API (internal/session/auth.go @ b68e498):
//
//	session.NewSessionAuth() *SessionAuth
//	(*SessionAuth).RegisterKey(sessionName string, key ConsoleKey, role Role)
//	(*SessionAuth).Authorize(key ConsoleKey, sessionName string) (Role, error)
//
// ConsoleKey is a string typedef (internal/session/fanout.go).
// The "router has no Tier-2 state" invariant (DI-010) is audited separately by
// TestSessionAuth_RouterHasNoTier2State (internal/session/auth_test.go:~217),
// which greps internal/routing/ source for Tier2/TierTwo identifiers.
func TestPropSessionAuth_RejectsUnauthorizedKey(t *testing.T) {
	properties := gopter.NewProperties(nil)

	properties.Property("Authorize returns ErrSessionAuthDenied for key not in auth list", prop.ForAll(
		func(listSize int) bool {
			listSize = listSize % 10 // 0–9 registered entries
			const testSession = "session-prop-test"

			sa := session.NewSessionAuth()
			for i := 0; i < listSize; i++ {
				// Register a set of distinct keys under the test session.
				registered := session.ConsoleKey(fmt.Sprintf("registered-key-%d", i))
				sa.RegisterKey(testSession, registered, session.RoleFull)
			}

			// A key that was never registered must be denied.
			unregistered := session.ConsoleKey("unregistered-console-key")
			_, err := sa.Authorize(unregistered, testSession)
			return errors.Is(err, session.ErrSessionAuthDenied)
		},
		gen.IntRange(0, 10),
	))

	properties.TestingRun(t)
}
```
