# VP-069: mgmt.Server.Serve Returns nil on Intentional Shutdown or Context Cancellation; Non-nil on Unexpected Listener Failure

## Property Statement

`mgmt.Server.Serve(ctx context.Context) error` returns `nil` — not an error — when
the server stops due to either of these intentional shutdown paths:

1. `mgmt.Server.Shutdown(ctx)` is called on the same server instance.
2. The context `ctx` passed to `Serve` is cancelled.

And returns a **non-nil** error on unexpected fatal listener failure — specifically,
when the underlying listener fd is closed externally (e.g., direct `ln.Close()` from
outside `Serve`) while:
- The context `ctx` is still alive (not cancelled), AND
- `Shutdown` has never been called (`shuttingDown` is false).

This is the "unexpected-close" path. A management plane that dies silently in this
condition violates SOUL #4 (no silent failure) and BC-2.07.004 PC-10.

The distinction is mediated by a `shuttingDown atomic.Bool` field: `Shutdown` and
the ctx-watcher goroutine each set `shuttingDown = true` before calling `s.ln.Close()`.
The canonical Accept error predicate is:
```go
if s.shuttingDown.Load() || (errors.Is(err, net.ErrClosed) && ctx.Err() != nil) {
    return nil  // intentional shutdown
}
return err  // unexpected listener close — non-nil
```
Both conjuncts of the `errors.Is` arm are required: `ctx.Err() != nil` ensures that
an externally-closed fd with a live context is NOT treated as intentional shutdown.
The `s.shuttingDown.Load()` arm handles the case where `Shutdown` is called before
the listener error is observed (flag set before `ln.Close()`, race-free). All other
Accept errors propagate as non-nil.

**Fatal-accept-error drain (Ruling P — BC-2.07.004 PC-10 extension):** On the
fatal-accept-error path (the predicate above evaluates to false — i.e., ctx is live,
Shutdown never called, but Accept returned a non-transient error), `s.closeAllConns()`
MUST be called immediately before `s.connWG.Wait()` before returning the error. This
ensures in-flight connection goroutines are force-closed and drain completes within
the caller's shutdown budget (not blocked up to `RPCIdleTimeout` = 30 s). Verified
by `TestServe_FatalAcceptErrorDrainsQuickly`: a stalled authenticated connection must
drain within 200 ms of an unexpected listener close, not within 30 s.

**Scope of "ALL paths" (Ruling P precision):** The fatal-accept-error return is the
gap Ruling P closes — `closeAllConns()` MUST precede `connWG.Wait()` on this path.
The ctx-cancel and Shutdown-initiated paths are excluded: `closeAllConns()` is already
called by `Shutdown()` on those paths. Pass-1 and pass-2 adversaries independently
confirmed the ctx-cancel arms are a benign ordering condition, not a hang; `go test
-race` passes. This is a spec-wording precision, no code change.

## Proof Harness Skeleton

```go
package mgmt_test

import (
    "context"
    "crypto/ed25519"
    "crypto/rand"
    "net"
    "testing"
    "time"

    "github.com/arcavenae/switchboard/internal/mgmt"
)

// TestServe_ReturnsNilOnShutdown_VP069 verifies VP-069:
// Serve returns nil on intentional shutdown and non-nil on unexpected failure.
func TestServe_ReturnsNilOnShutdown_VP069(t *testing.T) {
    t.Parallel()

    makeServer := func(t *testing.T) (*mgmt.Server, net.Listener) {
        t.Helper()
        ln, err := net.Listen("tcp", "127.0.0.1:0")
        if err != nil {
            t.Fatalf("listen: %v", err)
        }
        _, key, _ := ed25519.GenerateKey(rand.Reader)
        srv := mgmt.NewServer(ln, key, nil, nil, "dev")
        return srv, ln
    }

    t.Run("shutdown_returns_nil", func(t *testing.T) {
        t.Parallel()
        srv, _ := makeServer(t)
        ctx := context.Background()

        errCh := make(chan error, 1)
        go func() { errCh <- srv.Serve(ctx) }()

        time.Sleep(5 * time.Millisecond) // let Serve enter Accept
        if err := srv.Shutdown(context.Background()); err != nil {
            t.Fatalf("Shutdown: %v", err)
        }

        select {
        case err := <-errCh:
            if err != nil {
                t.Errorf("VP-069 violated: Serve returned non-nil after Shutdown: %v", err)
            }
        case <-time.After(2 * time.Second):
            t.Error("VP-069: Serve did not return within 2s of Shutdown")
        }
    })

    t.Run("ctx_cancel_returns_nil", func(t *testing.T) {
        t.Parallel()
        srv, _ := makeServer(t)
        ctx, cancel := context.WithCancel(context.Background())

        errCh := make(chan error, 1)
        go func() { errCh <- srv.Serve(ctx) }()

        time.Sleep(5 * time.Millisecond)
        cancel()

        select {
        case err := <-errCh:
            if err != nil {
                t.Errorf("VP-069 violated: Serve returned non-nil after ctx cancel: %v", err)
            }
        case <-time.After(2 * time.Second):
            t.Error("VP-069: Serve did not return within 2s of ctx cancel")
        }
    })

    // TestServe_ReturnsErrOnUnexpectedListenerClose (Ruling G — unexpected-close path)
    // Verifies the third VP-069 clause: when the listener fd is closed externally
    // (not via Shutdown, not via ctx cancel), Serve MUST return non-nil.
    // This test fails under code missing the "&& ctx.Err() != nil" conjunct.
    t.Run("unexpected_close_returns_non_nil", func(t *testing.T) {
        t.Parallel()
        srv, ln := makeServer(t)
        ctx := context.Background() // live context — never cancelled

        errCh := make(chan error, 1)
        go func() { errCh <- srv.Serve(ctx) }()

        time.Sleep(5 * time.Millisecond) // let Serve enter Accept
        // Close the listener directly — do NOT call Shutdown, do NOT cancel ctx.
        _ = ln.Close()

        select {
        case err := <-errCh:
            if err == nil {
                t.Error("VP-069 violated (Ruling G): Serve returned nil on unexpected listener close; expected non-nil")
            }
            // Any non-nil error satisfies the property — the exact error is not constrained.
        case <-time.After(2 * time.Second):
            t.Error("VP-069: Serve did not return within 2s of unexpected listener close")
        }
    })
}
```
