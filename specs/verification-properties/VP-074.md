# VP-074: QualityIndicator Threshold Classification Maps (RTT, Loss) â†’ (Green | Yellow | Red) Correctly

## Property Statement

For each (RTT ms, loss pct) pair the `QualityIndicator` must produce the
exact Quality level specified by BC-2.06.001 PC-2/PC-3/PC-4:

1. Green: RTT p99 â‰¤ 100 ms AND loss â‰¤ 5 %
2. Yellow: RTT p99 âˆˆ (100, 500] ms OR loss âˆˆ (5, 20] %
3. Red: RTT p99 > 500 ms OR loss > 20 % OR no paths available

Boundary values (RTT = 100, 100.001, 500, 500.001; loss = 5, 5.001, 20, 20.001)
must each resolve to the correct side of the boundary. Exactly three enum
values exist: `Green`, `Yellow`, `Red`; no other value is ever returned.

- **BC:** BC-2.06.001 â€” Quality Indicator (green/yellow/red) Derived from Measured Path Latency and Loss
- **Postconditions:** PC-2 (Green threshold), PC-3 (Yellow threshold), PC-4 (Red threshold)

## Proof Harness Skeleton

```go
package metrics_test

import (
	"testing"

	"github.com/arcavenae/switchboard/internal/metrics"
)

func TestQualityIndicator_ThresholdClassification(t *testing.T) {
	t.Parallel()

	cases := []struct {
		name    string
		rttMs   float64
		lossPct float64
		want    metrics.Quality
	}{
		// --- nominal green ---
		{"green_rtt_low_loss_zero", 15, 0, metrics.Green},
		{"green_rtt_mid_loss_low", 50, 3, metrics.Green},
		// --- green/yellow boundaries ---
		{"boundary_rtt_100_green", 100.0, 0, metrics.Green},
		{"boundary_rtt_100001_yellow", 100.001, 0, metrics.Yellow},
		{"boundary_loss_5_green", 10, 5.0, metrics.Green},
		{"boundary_loss_5001_yellow", 10, 5.001, metrics.Yellow},
		// --- nominal yellow ---
		{"yellow_rtt_only", 200, 0, metrics.Yellow},
		{"yellow_loss_only", 10, 10, metrics.Yellow},
		{"yellow_rtt_and_loss", 150, 3, metrics.Yellow},
		// --- yellow/red boundaries ---
		{"boundary_rtt_500_yellow", 500.0, 0, metrics.Yellow},
		{"boundary_rtt_500001_red", 500.001, 0, metrics.Red},
		{"boundary_loss_20_yellow", 10, 20.0, metrics.Yellow},
		{"boundary_loss_20001_red", 10, 20.001, metrics.Red},
		// --- nominal red ---
		{"red_rtt_high", 600, 25, metrics.Red},
	}

	for _, tc := range cases {
		tc := tc
		t.Run(tc.name, func(t *testing.T) {
			t.Parallel()
			qi := metrics.NewQualityIndicator()
			// Downgrade is immediate (BC-2.06.001 Invariant 3 / AC-004).
			// A single update is sufficient to assert classification.
			qi.Update(tc.rttMs, tc.lossPct)
			if got := qi.Current(); got != tc.want {
				t.Errorf("VP-074: Update(%.3f ms, %.3f%%): got %v, want %v",
					tc.rttMs, tc.lossPct, got, tc.want)
			}
		})
	}
}
```
