# VP-034: Multi-Console Fan-Out

## Property Statement

With two console nodes attached to the same session:
  ∀ downstream frame F emitted by the access node:
    console_1 receives F ∧ console_2 receives F

The access node must fan out every downstream frame to all attached consoles.
Neither console may be starved when both are simultaneously attached.

## Source Contract

- **BC:** BC-2.04.006 — Two or more consoles may subscribe to the same session output simultaneously
- **Postcondition/Invariant:** Postcondition — "All attached consoles receive all downstream frames."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| e2e | go test (testing.T) | yes | two consoles attached to same session; generate N frames; assert both consoles receive all N |

## Proof Harness Skeleton

```go
//go:build integration

package session_test

import (
	"context"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/testenv"
)

// TestE2E_MultiConsole_FanOut verifies that two simultaneously attached
// consoles both receive all downstream frames from a session.
func TestE2E_MultiConsole_FanOut(t *testing.T) {
	t.Parallel()

	ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
	t.Cleanup(cancel)

	env := testenv.New(t, ctx)
	t.Cleanup(env.Close)

	sessionID := env.CreateSession(t)

	console1 := env.AttachConsole(t, sessionID)
	console2 := env.AttachConsole(t, sessionID)

	// Generate downstream output.
	const lines = 5
	for i := 0; i < lines; i++ {
		env.SendKeystroke(t, sessionID, "echo line\n")
		time.Sleep(50 * time.Millisecond)
	}

	// Allow frames to propagate.
	time.Sleep(500 * time.Millisecond)

	frames1 := console1.CollectFrames(t, 0)
	frames2 := console2.CollectFrames(t, 0)

	if len(frames1) == 0 {
		t.Error("console1 received no frames")
	}
	if len(frames2) == 0 {
		t.Error("console2 received no frames")
	}

	// Both consoles must have received the same number of frames.
	if len(frames1) != len(frames2) {
		t.Errorf("frame count mismatch: console1=%d console2=%d", len(frames1), len(frames2))
	}
}
```
