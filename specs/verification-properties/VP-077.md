# VP-077: Admin list-keys Admission-Gate — Any-Role OR Operator-Set OR Bootstrap-Key

> Authored 2026-07-03 to close BC↔VP↔AC triangle gap identified as F-P5P14-B-003.
> Complementary to VP-075 (which scope-excludes list-keys from write-authority property
> scope). VP-075 verifies that mutating operations (register/revoke/expire) reject
> non-control callers. VP-077 verifies that list-keys enforces the admission gate —
> a distinct, orthogonal property covering the read path.

## Property Statement

For any caller context `c`, target SVTN `s`, and `admin.key.list-keys` request `r`:

```
result(admin.key.list-keys, c, s, r) = ok(payload)  iff
  IsAdmittedAnyRole(c, s)     -- key registered in s with any active role (control, console, access)
  OR c ∈ OperatorKeySet(s)    -- key is a member of the daemon operator-set
  OR c = BootstrapKey(s)      -- key is the daemon bootstrap pubkey

result(admin.key.list-keys, c, s, r) = E-ADM-009    otherwise
```

Additionally, when `s` does not exist (SVTN not found), `result = E-SVTN-003` regardless of
admission status (SVTN-existence check precedes the admission gate).

The three reachable admission-failure modes (BC-2.05.004 EC-008):

1. **No caller identity:** CallerPubkey absent from context AND no ambient bootstrap identity —
   handler cannot resolve caller; returns E-ADM-009.
2. **Cross-SVTN / unregistered caller:** CallerPubkey present but not registered on the target
   SVTN AND not in operator-set AND not the daemon bootstrap key — CWE-862 defense against
   cross-SVTN roster enumeration; returns E-ADM-009.
3. **Revoked or expired registration:** CallerPubkey present, registered on target SVTN, but
   `revoked=true` or `now >= expiry` — registered-any-state is insufficient; only active
   admission qualifies; returns E-ADM-009.

**Scope:** Covers BC-2.05.004 v1.14 EC-008 (the three admission-failure modes for
`admin.key.list-keys`). Does NOT cover write operations (`admin.key.register`,
`admin.key.revoke`, `admin.key.expire`) — those retain the control-role authority gate and
are covered by VP-075.

**Impl anchor:** `cmd/switchboard/admin_handlers.go:363` `resolveCallerAdmissionAnyRole`
(landed via F-P5P13-A-001 remediation, Burst 37, develop tip commit 03ce8e7).

## Source Contract

- **BC:** BC-2.05.004 v1.14 — Key Lifecycle — Register, Revoke, and Expire Admission and
  Session-Authorization Keys
- **EC-008:** Three reachable `admin.key.list-keys` admission failure modes (missing caller,
  cross-SVTN / unregistered, revoked/expired registration). Each returns E-ADM-009.
- **Precondition 1 — list-keys authority (F-L2-003):** The control-only AUTHORITY gate is
  removed for list-keys; the ADMISSION gate remains. Callers must be admitted to the target
  SVTN in any active role, OR be a member of the operator-set, OR be the daemon bootstrap key.
- **CWE-862 / CWE-200:** Cross-SVTN callers must not enumerate another SVTN's admitted roster.
  The admission gate is the enforcement mechanism.
- **Capability Anchor:** CAP-019 ("Key lifecycle management (register, revoke, expire)")
  per capabilities.md §CAP-019.

## Proof Harness Skeleton

```go
//go:build integration

package main

import (
	"context"
	"crypto/ed25519"
	"crypto/rand"
	"net"
	"path/filepath"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/admission"
	"github.com/arcavenae/switchboard/internal/mgmt"
	"github.com/arcavenae/switchboard/internal/svtnmgmt"
)

// TestIntegration_AdminListKeys_AdmissionGate_VP077 verifies VP-077:
// admin.key.list-keys enforces the admission gate (any-role OR operator-set
// OR bootstrap-key; else E-ADM-009). Complementary to VP-075 which covers
// the write-authority gate on mutating operations.
//
// Helper functions sendAdminRPC / readAdminRPCResponse / authenticateAdminClient
// follow the pattern in cmd/switchboard/admin_handlers_e2e_test.go.
func TestIntegration_AdminListKeys_AdmissionGate_VP077(t *testing.T) {
	t.Parallel()

	// Generate a keypair that will NOT be registered in the target SVTN.
	unregisteredPub, unregisteredPriv, err := ed25519.GenerateKey(rand.Reader)
	if err != nil {
		t.Fatalf("generate unregistered key: %v", err)
	}

	// Generate a console-role keypair that will be admitted.
	consolePub, consolePriv, err := ed25519.GenerateKey(rand.Reader)
	if err != nil {
		t.Fatalf("generate console key: %v", err)
	}

	svtnMgr := svtnmgmt.New()
	if _, err := svtnMgr.Create("test-svtn"); err != nil {
		t.Fatalf("create test-svtn: %v", err)
	}
	// Register consolePub as admitted (any role satisfies the admission gate).
	if _, err := svtnMgr.RegisterKey("test-svtn", consolePub, admission.RoleConsole); err != nil {
		t.Fatalf("register console key: %v", err)
	}

	// Operator-set includes the unregistered key so auth can complete;
	// the admission gate fires after auth at the handler layer.
	ops := mgmt.NewOperatorKeySet([]ed25519.PublicKey{unregisteredPub})
	handlers := BuildAdminHandlers(svtnMgr, ops)

	subCases := []struct {
		name      string
		keyPriv   ed25519.PrivateKey
		expectOK  bool
		expectErr string // error code if expectOK == false
	}{
		// Cross-SVTN / unregistered caller — not admitted; expects E-ADM-009
		{"unregistered_cross_svtn_denied", unregisteredPriv, false, "E-ADM-009"},
		// Admitted console-role caller — expects ok: true
		{"admitted_console_role_allowed", consolePriv, true, ""},
	}

	for _, tc := range subCases {
		tc := tc
		t.Run(tc.name, func(t *testing.T) {
			t.Parallel()

			sockDir := t.TempDir()
			sockPath := filepath.Join(sockDir, "mgmt.sock")
			ln, err := net.Listen("unix", sockPath)
			if err != nil {
				t.Fatalf("listen: %v", err)
			}
			t.Cleanup(func() { ln.Close() })

			_, daemonPriv, err := ed25519.GenerateKey(rand.Reader)
			if err != nil {
				t.Fatalf("generate daemon key: %v", err)
			}

			ctx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
			t.Cleanup(cancel)
			go func() {
				_ = mgmt.NewServer(ln, daemonPriv, ops, handlers, "dev").Serve(ctx)
			}()

			conn, err := net.Dial("unix", sockPath)
			if err != nil {
				t.Fatalf("dial: %v", err)
			}
			t.Cleanup(func() { conn.Close() })

			authenticateAdminClient(t, conn, tc.keyPriv, daemonPriv.Public().(ed25519.PublicKey))

			sendAdminRPC(t, conn, "admin.key.list-keys", map[string]interface{}{
				"svtn_id": "test-svtn",
			})
			resp := readAdminRPCResponse(t, conn)

			if tc.expectOK {
				if resp["ok"] != true {
					t.Errorf("expected ok:true, got ok:%v (code:%v)", resp["ok"], resp["code"])
				}
			} else {
				if resp["ok"] != false {
					t.Errorf("expected ok:false, got ok:%v", resp["ok"])
				}
				code, _ := resp["code"].(string)
				if code != tc.expectErr {
					t.Errorf("expected code:%q, got code:%q", tc.expectErr, code)
				}
			}
		})
	}
}
```
