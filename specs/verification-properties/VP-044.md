# VP-044: Presence Advertisement Includes Required Fields

## Property Statement

A PresenceAdvertisement message emitted by an access node:
  session_name is non-empty
  ∧ attachment_status is non-empty
  ∧ quality_indicator is non-empty

These three fields are required by the discovery protocol. A missing or
empty field prevents consoles from displaying meaningful session information
and is a protocol violation.

## Source Contract

- **BC:** BC-2.03.001 — Access node advertises session presence via SVTN-scoped multicast on state change and periodic heartbeat
- **BC:** BC-2.03.003 — Presence advertisement includes session name, attachment status, and quality indicator
- **Postcondition/Invariant:** Postcondition — "PresenceAdvertisement contains non-empty session_name, attachment_status, quality_indicator."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| integration | go test (testing.T) | yes | start access node; capture first PresenceAdvertisement; assert required fields present and non-empty |

## Proof Harness Skeleton

```go
//go:build integration

package discovery_test

import (
	"context"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/discovery"
	"github.com/arcavenae/switchboard/internal/testenv"
)

// TestIntegration_PresenceAdvertisement_RequiredFields verifies that an
// access node's PresenceAdvertisement contains all required fields.
func TestIntegration_PresenceAdvertisement_RequiredFields(t *testing.T) {
	t.Parallel()

	ctx, cancel := context.WithTimeout(context.Background(), 15*time.Second)
	t.Cleanup(cancel)

	env := testenv.New(t, ctx)
	t.Cleanup(env.Close)

	env.CreateSession(t)

	// Subscribe to presence advertisements.
	advCh := make(chan discovery.PresenceAdvertisement, 8)
	sub := env.SubscribePresence(t, func(adv discovery.PresenceAdvertisement) {
		advCh <- adv
	})
	t.Cleanup(sub.Cancel)

	// Wait for at least one advertisement.
	var adv discovery.PresenceAdvertisement
	select {
	case adv = <-advCh:
	case <-ctx.Done():
		t.Fatal("timed out waiting for PresenceAdvertisement")
	}

	if adv.SessionName == "" {
		t.Error("PresenceAdvertisement.SessionName is empty")
	}
	if adv.AttachmentStatus == "" {
		t.Error("PresenceAdvertisement.AttachmentStatus is empty")
	}
	if adv.QualityIndicator == "" {
		t.Error("PresenceAdvertisement.QualityIndicator is empty")
	}
}
```
