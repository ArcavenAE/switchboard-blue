# VP-027: QualityIndicator Degradation Transitions Only Monotonically Downward

## Property Statement

∀ sustained degradation sequence (RTT rising, loss rising):
  quality transitions follow only: green → yellow → red
  ∧ no single-step red → green transition occurs without an intermediate yellow

Under a monotone degradation workload the quality indicator must not
improve without first passing through an intermediate state. The
red → green skip is explicitly forbidden; recovery must go red → yellow → green.

## Source Contract

- **BC:** BC-2.06.001 — QualityIndicator Green/Yellow/Red Thresholds
- **BC:** BC-2.06.002 — QualityIndicator Transition Hysteresis
- **Postcondition/Invariant:** Invariant — "Transitions during sustained degradation are monotonically downward."

## Proof Harness Skeleton

```go
package metrics_test

import (
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/metrics"
)

// genDegradationSequence produces a sequence of (rtt_ms, loss_pct) pairs
// where each successive value is >= the previous (monotone non-decreasing).
func genDegradationSequence() gopter.Gen {
	return gen.SliceOfN(16, gopter.CombineGens(
		gen.UInt32Range(0, 50),   // rtt increment
		gen.UInt8Range(0, 5),     // loss increment percent
	)).Map(func(vals interface{}) []metrics.Sample {
		pairs := vals.([]interface{})
		samples := make([]metrics.Sample, len(pairs))
		var rttAcc uint32
		var lossAcc uint32
		for i, v := range pairs {
			pair := v.([]interface{})
			rttAcc += pair[0].(uint32)
			lossAcc += uint32(pair[1].(uint8))
			if lossAcc > 100 {
				lossAcc = 100
			}
			samples[i] = metrics.Sample{RTTMs: rttAcc, LossPct: lossAcc}
		}
		return samples
	})
}

func TestProp_QualityIndicator_MonotoneDownwardDuringDegradation(t *testing.T) {
	properties := gopter.NewProperties(nil)

	properties.Property("no red→green skip during monotone degradation", prop.ForAll(
		func(samples []metrics.Sample) bool {
			qi := metrics.NewQualityIndicator(metrics.DefaultThresholds())
			prevState := qi.State()

			for _, s := range samples {
				qi.Update(s)
				cur := qi.State()
				// Detect illegal red→green transition.
				if prevState == metrics.QualityRed && cur == metrics.QualityGreen {
					return false
				}
				prevState = cur
			}
			return true
		},
		genDegradationSequence(),
	))

	properties.TestingRun(t)
}
```
