# VP-037: Router Drain — Nodes Migrate Within 2s

## Property Statement

After a router sends DRAIN_SIGNAL:
  connected nodes connect to an alternate router within 2 seconds
  ∧ session traffic resumes on the alternate path

No session should be permanently lost due to a router drain. The 2-second
bound ensures operator-initiated maintenance windows are bounded.

## Source Contract

- **BC:** BC-2.09.002 — Router sends drain signal before shutdown; nodes migrate to alternate routers
- **Postcondition/Invariant:** Postcondition — "All connected nodes migrate to alternate router ≤2s after DRAIN_SIGNAL."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| e2e | go test (testing.T) | yes | two-router stack; send DRAIN_SIGNAL on router1; measure time until session traffic resumes on router2 |

## Proof Harness Skeleton

```go
//go:build integration

package drain_test

import (
	"context"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/testenv"
)

// TestE2E_RouterDrain_NodesMigrateWithin2s verifies that nodes connected to
// a draining router migrate to an alternate router within 2 seconds.
func TestE2E_RouterDrain_NodesMigrateWithin2s(t *testing.T) {
	t.Parallel()

	ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
	t.Cleanup(cancel)

	// Two-router topology.
	env := testenv.NewWithRouters(ctx, t, 2) // corrected v1.4 — F-DW-SP1-012: actual signature is (ctx, t, n)
	t.Cleanup(env.Close)

	sessionID := env.CreateSession(t)

	// Confirm traffic is flowing.
	env.SendKeystroke(t, sessionID, "echo pre-drain\n")
	pre := env.CollectFrames(t, sessionID, 2*time.Second)
	if len(pre) == 0 {
		t.Fatal("no frames before drain signal; check stack setup")
	}

	// Send drain signal on router 0.
	drainAt := time.Now()
	env.SendDrainSignal(t, 0)

	// Poll until frames resume on the alternate router, up to 2s.
	const maxMigration = 2 * time.Second
	deadline := drainAt.Add(maxMigration)
	var resumed bool
	for time.Now().Before(deadline) {
		env.SendKeystroke(t, sessionID, "echo post-drain\n")
		post := env.CollectFrames(t, sessionID, 200*time.Millisecond)
		if len(post) > 0 {
			resumed = true
			break
		}
	}

	if !resumed {
		t.Errorf("session traffic did not resume within %v after DRAIN_SIGNAL", maxMigration)
	}
}
```
