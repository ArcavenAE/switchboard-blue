# VP-024: Multipath Delivers First Copy, Discards Duplicates

## Property Statement

∀ two frames f1, f2 where checksum(f1) == checksum(f2):
  arriving in order f1 then f2:
    f1 is passed to deliver()
    f2 is silently discarded (deliver() not called for f2)

The first copy of any frame (identified by checksum) is delivered; any
subsequent copy with the same checksum is dropped without delivery and
without side effects.

## Source Contract

- **BC:** BC-2.02.001 — Duplicate-and-race: same frame sent on two fastest paths simultaneously
- **BC:** BC-2.02.002 — Receiver delivers first-arriving copy and silently discards subsequent duplicates
- **Postcondition/Invariant:** Postcondition — "Second and subsequent copies of a frame are silently discarded."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| proptest | gopter v0.2.9+ | no | random frame payloads; inject original then duplicate; assert deliver called exactly once |

## Proof Harness Skeleton

```go
package multipath_test

import (
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/frame"
	"github.com/arcavenae/switchboard/internal/multipath"
)

func TestProp_Multipath_DeliverFirstDiscardDuplicate(t *testing.T) {
	properties := gopter.NewProperties(nil)

	properties.Property("first copy delivered; duplicate silently discarded", prop.ForAll(
		func(payload []byte, extraCopies uint8) bool {
			if extraCopies == 0 {
				extraCopies = 1
			}
			deliverCount := 0
			mp := multipath.New(multipath.Config{DropCacheSize: 256})
			mp.OnDeliver(func(f frame.Frame) {
				deliverCount++
			})

			original := frame.Frame{}
			original.Payload = payload
			original.Checksum = frame.ComputeChecksum(original)

			// Deliver original.
			mp.OnReceive(original)

			// Deliver N duplicate copies.
			for i := 0; i < int(extraCopies); i++ {
				dup := original
				mp.OnReceive(dup)
			}

			return deliverCount == 1
		},
		gen.SliceOf(gen.UInt8()),
		gen.UInt8Range(1, 5),
	))

	properties.TestingRun(t)
}
```
