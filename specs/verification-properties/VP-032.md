# VP-032: PTY Fallback Activates on Control Mode Failure

## Property Statement

When the tmux control mode socket is unavailable:
  access node activates PTY proxy mode
  ∧ a fallback event is logged at INFO or WARN level

The access node must not crash or enter an undefined state when tmux
control mode is unreachable. It must fall back to PTY proxying transparently
and record a log entry describing the fallback for operator visibility.

## Source Contract

- **BC:** BC-2.04.002 — Access node falls back to PTY proxy when tmux control mode unavailable
- **Postcondition/Invariant:** Postcondition — "Access node activates PTY proxy and logs fallback when control mode socket unavailable."

## Proof Harness Skeleton

```go
//go:build integration

package tmux_test

import (
	"bytes"
	"context"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/tmux"
)

// TestTmux_PTYFallback_OnControlModeFailure verifies that the access node
// falls back to PTY proxy mode when the tmux control mode socket is absent.
func TestTmux_PTYFallback_OnControlModeFailure(t *testing.T) {
	t.Parallel()

	ctx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
	t.Cleanup(cancel)

	var logBuf bytes.Buffer

	// Configure with a non-existent socket path to force control mode failure.
	cfg := tmux.Config{
		SocketPath: "/tmp/nonexistent-switchboard-tmux-test.sock",
		Logger:     tmux.NewBufferLogger(&logBuf),
	}

	node, err := tmux.NewAccessNode(ctx, cfg)
	if err != nil {
		// Node construction itself should not fail — fallback happens internally.
		t.Fatalf("NewAccessNode returned unexpected error: %v", err)
	}
	t.Cleanup(func() { _ = node.Close() })

	// Give the node time to attempt control mode and fall back.
	time.Sleep(500 * time.Millisecond)

	// Assert PTY mode is active.
	if node.Mode() != tmux.ModePTY {
		t.Errorf("expected PTY mode after control mode failure, got %v", node.Mode())
	}

	// Assert fallback was logged.
	if !bytes.Contains(logBuf.Bytes(), []byte("fallback")) {
		t.Errorf("expected 'fallback' in log output; got: %s", logBuf.String())
	}
}
```
