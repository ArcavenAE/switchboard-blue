# VP-021: ARQ.TLPKTDROP Triggers Only When Frame Is Overdue

## Property Statement

∀ frame f sent at time t0 with tlpktdrop_timeout T:
  TLPKTDROP fires ⟺ (current_time - t0) > T ∧ f is still outstanding

TLPKTDROP must not fire before the timeout elapses. It must fire (at most
once per frame) once the timeout has elapsed and the frame has not been acked.
If the frame is acked before the timeout, TLPKTDROP must not fire for it.

## Source Contract

- **BC:** BC-2.02.006 — TLPKTDROP terminates overdue downstream frames and signals degradation
- **Postcondition/Invariant:** Invariant — "TLPKTDROP fires if and only if frame outstanding for > tlpktdrop_timeout."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| proptest | gopter v0.2.9+ | no | random timeout values and event timings; simulated clock; asserts early-fire and miss-fire absence |

## Proof Harness Skeleton

```go
package arq_test

import (
	"testing"
	"time"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/arq"
	"github.com/arcavenae/switchboard/internal/frame"
)

func TestProp_ARQ_TLPKTDROPFiresOnlyWhenOverdue(t *testing.T) {
	properties := gopter.NewProperties(nil)

	// Case 1: ack arrives before timeout — TLPKTDROP must not fire.
	properties.Property("no TLPKTDROP when ack arrives before timeout", prop.ForAll(
		func(timeoutMs uint16, ackOffsetMs uint8) bool {
			if timeoutMs == 0 {
				timeoutMs = 1
			}
			timeout := time.Duration(timeoutMs) * time.Millisecond
			// ackOffset is strictly less than timeout.
			ackOffset := time.Duration(ackOffsetMs%uint8(timeoutMs)) * time.Millisecond

			dropped := false
			clk := arq.NewSimClock()
			a := arq.New(arq.Config{
				WindowSize:       8,
				TLPKTDROPTimeout: timeout,
				Clock:            clk,
			})
			a.OnTLPKTDROP(func(seq uint64) { dropped = true })

			f := frame.Frame{}
			f.Sequence = 0
			_ = a.Send(f)

			clk.Advance(ackOffset)
			a.OnAck(0)
			clk.Advance(timeout + time.Millisecond) // advance past timeout after ack
			a.Tick()

			return !dropped
		},
		gen.UInt16Range(1, 500),
		gen.UInt8(),
	))

	// Case 2: no ack arrives within timeout — TLPKTDROP must fire.
	properties.Property("TLPKTDROP fires when frame outstanding past timeout", prop.ForAll(
		func(timeoutMs uint16) bool {
			if timeoutMs == 0 {
				timeoutMs = 1
			}
			timeout := time.Duration(timeoutMs) * time.Millisecond

			dropped := false
			clk := arq.NewSimClock()
			a := arq.New(arq.Config{
				WindowSize:       8,
				TLPKTDROPTimeout: timeout,
				Clock:            clk,
			})
			a.OnTLPKTDROP(func(seq uint64) { dropped = true })

			f := frame.Frame{}
			f.Sequence = 0
			_ = a.Send(f)

			clk.Advance(timeout + time.Millisecond)
			a.Tick()

			return dropped
		},
		gen.UInt16Range(1, 500),
	))

	properties.TestingRun(t)
}
```
