# VP-011: Split-Horizon: No Forward Toward Arrival Interface

## Property Statement

∀ frame f arriving on interface I,
  routing_table RoutingTable:
  I ∉ SVTNRoute(routing_table, f, arrivalInterface=I).OutputInterfaces

The arrival interface must never appear in the set of output interfaces
returned by SVTNRoute. This holds even when the routing table's forwarding
entry for the destination lists I as the best path. The split-horizon rule
is stateless per-frame and applies to all SVTN frame types.

## Source Contract

- **BC:** BC-2.02.008 — Router split-horizon prevents frames being forwarded back toward arrival interface
- **Postcondition/Invariant:** Postcondition 1 — "The frame is not forwarded back on the arrival interface, regardless of what the forwarding table says." Invariant 1 — "Split-horizon applies to all SVTN frame types."

## Proof Harness Skeleton

```go
package routing_test

import (
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/frame"
	"github.com/arcavenae/switchboard/internal/routing"
)

func TestPropSplitHorizon_ArrivalInterfaceNotInOutput(t *testing.T) {
	properties := gopter.NewProperties(nil)

	properties.Property("arrival interface never appears in SVTNRoute output interfaces", prop.ForAll(
		func(ifaceCount int, arrivalIdx int) bool {
			if ifaceCount < 1 {
				return true
			}
			ifaceCount = (ifaceCount % 7) + 1   // 1–8 interfaces
			arrivalIdx = arrivalIdx % ifaceCount

			// Build a routing table that deliberately lists all interfaces
			// for the destination — worst case for split-horizon.
			table := routing.NewTable()
			allIfaces := make([]routing.InterfaceID, ifaceCount)
			for i := range allIfaces {
				allIfaces[i] = routing.InterfaceID(i)
			}

			var svtnID [16]byte
			var dstAddr [8]byte
			dstAddr[0] = 0xAA
			table.AddRoute(svtnID, dstAddr, allIfaces)

			arrivalIface := allIfaces[arrivalIdx]

			f := frame.Frame{}
			f.OuterHeader.SVTNID = svtnID
			f.OuterHeader.DstAddr = dstAddr

			result := routing.SVTNRoute(table, f, arrivalIface)

			for _, out := range result.OutputInterfaces {
				if out == arrivalIface {
					return false // split-horizon violated
				}
			}
			return true
		},
		gen.IntRange(1, 8),
		gen.IntRange(0, 100),
	))

	properties.TestingRun(t)
}
```
