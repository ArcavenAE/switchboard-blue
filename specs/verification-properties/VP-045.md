# VP-045: Console Session Enumeration Without Hostnames

## Property Statement

A console node with no hostnames or IP addresses of access nodes configured:
  can retrieve a non-empty session list via discovery

Zero-configuration session discovery is a first-class feature. Consoles
must not require manual IP/hostname configuration to discover available
sessions on the local network.

## Source Contract

- **BC:** BC-2.03.002 — Console enumerates all SVTN sessions without specifying hostnames or IP addresses
- **Postcondition/Invariant:** Postcondition — "Console retrieves non-empty session list without configuring access node addresses."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| e2e | go test (testing.T) | yes | access node with active session; console with no static addresses configured; list sessions via discovery API; assert non-empty list |

## Proof Harness Skeleton

```go
//go:build integration

package discovery_test

import (
	"context"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/discovery"
	"github.com/arcavenae/switchboard/internal/testenv"
)

// TestE2E_Console_SessionEnumeration_NoHostnames verifies that a console
// node can discover sessions without static access node addresses.
func TestE2E_Console_SessionEnumeration_NoHostnames(t *testing.T) {
	t.Parallel()

	ctx, cancel := context.WithTimeout(context.Background(), 20*time.Second)
	t.Cleanup(cancel)

	env := testenv.New(t, ctx)
	t.Cleanup(env.Close)

	// Create a session so there is something to discover.
	env.CreateSession(t)

	// Create a console with zero static configuration.
	console := env.NewConsoleNoStaticAddrs(t)

	// Allow discovery advertisements to propagate.
	time.Sleep(2 * time.Second)

	// List sessions via the zero-config discovery client.
	sessions, err := console.ListSessions(ctx, discovery.ListOptions{})
	if err != nil {
		t.Fatalf("ListSessions error: %v", err)
	}
	if len(sessions) == 0 {
		t.Error("expected at least one session in discovery list; got none")
	}
}
```
