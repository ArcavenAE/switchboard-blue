# VP-073: Console-Mode TCP Bound to Non-Loopback Address Aborts Startup with E-CFG-008

## Property Statement

When the daemon is running in console mode and the `management_socket` config value
specifies a host that is NOT `127.0.0.1`, `[::1]`, or `localhost`, the function
`buildMgmtListener` in `cmd/switchboard/mgmt_wire.go` MUST:

1. Return a non-nil error whose string representation contains `"E-CFG-008"`.
   The canonical full message prefix for this variant is:
   `"E-CFG-008: management_socket: console mode requires a loopback address (127.0.0.1, [::1], or localhost); got: <address>"`
   where `<address>` is the value from the config field. The error code `E-CFG-008`
   IS embedded as a prefix in the format string (grep-able in logs and test output).
   **Test assertion MUST use `strings.Contains(err.Error(), "E-CFG-008")`** — not a
   full-string match — so assertions remain correct if minor message wording changes.
2. Cause daemon startup to abort (non-zero exit). No management listener is opened.
3. This check MUST be in `buildMgmtListener`, NOT in `config.Validate()`.

Rejected hosts include (at minimum): `0.0.0.0`, `::` (IPv6 unspecified), bare port
(`:9091`, empty host), and any non-loopback IP.

Accepted hosts: `127.0.0.1`, `[::1]`, `localhost` (any port combination).

This extends the EC-013 loopback-rejection predicate added by Ruling D. The canonical
message string was clarified by Ruling L (ARCH-12 v1.4); the `buildMgmtListener`
variant embeds the code as prefix; the `config.Validate()` empty-socket variant does
not embed the code in the string.

## Proof Harness Skeleton

```go
package switchboard_test

import (
    "strings"
    "testing"

    switchboard "github.com/arcavenae/switchboard/cmd/switchboard"
)

// TestBuildMgmtListener_LoopbackRejection_VP073 verifies VP-073:
// buildMgmtListener aborts with E-CFG-008 for non-loopback console-mode addresses.
func TestBuildMgmtListener_LoopbackRejection_VP073(t *testing.T) {
    t.Parallel()

    type tc struct {
        addr    string
        wantErr bool
    }
    cases := []tc{
        // Rejected addresses.
        {"0.0.0.0:9091", true},
        {":::9091", true},       // bare IPv6 unspecified
        {":9091", true},         // empty host (bare port)
        {"192.168.1.1:9091", true},
        {"10.0.0.1:9091", true},
        // Accepted addresses.
        {"127.0.0.1:9091", false},
        {"[::1]:9091", false},
        {"localhost:9091", false},
    }

    for _, tc := range cases {
        tc := tc
        t.Run(tc.addr, func(t *testing.T) {
            t.Parallel()
            ln, err := switchboard.BuildMgmtListenerConsole(tc.addr)
            if ln != nil {
                ln.Close()
            }
            if tc.wantErr {
                if err == nil {
                    t.Errorf("VP-073 violated: expected E-CFG-008 error for %q, got nil", tc.addr)
                    return
                }
                if !strings.Contains(err.Error(), "E-CFG-008") {
                    t.Errorf("VP-073 violated: error does not contain E-CFG-008: %v", err)
                }
            } else {
                if err != nil {
                    t.Errorf("VP-073 violated: unexpected error for loopback %q: %v", tc.addr, err)
                }
            }
        })
    }
}
```
