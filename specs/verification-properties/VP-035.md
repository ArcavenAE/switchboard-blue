# VP-035: Read-Only Console Upstream Rejected

## Property Statement

A console with read-only role:
  upstream keystrokes are rejected by the access node with E-ADM-007
  (errors.Is(err, session.ErrUpstreamReadOnly))
  ∧ downstream frames continue to flow to the read-only console

The access node must enforce the read-only role: keystrokes sent upstream by
a read-only console must be rejected without terminating the session or
interrupting downstream delivery.

## Source Contract

- **BC:** BC-2.04.005 — Read-only console receives downstream stream; upstream keystrokes are rejected by access node
- **Postcondition/Invariant:** Postcondition — "Read-only console upstream rejected with E-ADM-007 (ErrUpstreamReadOnly); downstream unaffected."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| integration | go test (testing.T) | yes | attach read-only console; send upstream keystroke; assert rejection error code; assert downstream still flowing |

## Proof Harness Skeleton

```go
//go:build integration

package session_test

import (
	"context"
	"errors"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/session"
	"github.com/arcavenae/switchboard/internal/testenv"
)

// TestIntegration_ReadOnlyConsole_UpstreamRejected verifies that a read-only
// console cannot send upstream keystrokes but still receives downstream frames.
func TestIntegration_ReadOnlyConsole_UpstreamRejected(t *testing.T) {
	t.Parallel()

	ctx, cancel := context.WithTimeout(context.Background(), 15*time.Second)
	t.Cleanup(cancel)

	env := testenv.New(t, ctx)
	t.Cleanup(env.Close)

	sessionID := env.CreateSession(t)

	// Attach a read-only console.
	roConsole := env.AttachConsoleWithRole(t, sessionID, session.RoleReadOnly)

	// Attempt to send an upstream keystroke.
	err := roConsole.SendUpstream([]byte("echo blocked\n"))
	if err == nil {
		t.Fatal("expected upstream send to be rejected for read-only console, but it succeeded")
	}

	// Verify the error is ErrUpstreamReadOnly (E-ADM-007).
	if !errors.Is(err, session.ErrUpstreamReadOnly) {
		t.Errorf("expected ErrUpstreamReadOnly (E-ADM-007), got: %v", err)
	}

	// Downstream frames must still flow: generate output via a writable console.
	rwConsole := env.AttachConsole(t, sessionID)
	_ = rwConsole.SendUpstream([]byte("echo hello\n"))

	time.Sleep(500 * time.Millisecond)
	frames := roConsole.CollectFrames(t, 0)
	if len(frames) == 0 {
		t.Error("read-only console received no downstream frames after upstream rejection")
	}
}
```
