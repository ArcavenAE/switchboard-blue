# VP-028: Config.Validate Rejects Out-of-Range tick_interval

## Property Statement

∀ tick_interval d:
  d < 5ms ∨ d > 50ms
  ⟹ Validate(config) returns a non-empty error slice
      ∧ at least one error references field 'tick_interval_upstream' or 'tick_interval_downstream'

A tick interval outside the allowed range [5ms, 50ms] is a configuration
error that must be surfaced explicitly by field name, not silently clamped
or ignored.

## Source Contract

- **BC:** BC-2.09.003 — Router startup fails cleanly on malformed config with actionable error message
- **Postcondition/Invariant (derived):** BC-2.09.003 Postcondition 1 — "The daemon exits with a non-zero exit code before accepting any connections" when given a malformed config. This VP tests a specific malformed-config subcase: when `tick_interval` is outside [5ms, 50ms], `Config.Validate` must return an error naming the field (`tick_interval_upstream` or `tick_interval_downstream`) and the invalid value. The out-of-range tick_interval bound is config-validation territory per ASM-001 (tick interval bounds are operationally defined constraints). This VP is a DERIVED property: it specifies what "malformed config → exit non-zero" means concretely for the tick_interval field, not a postcondition literally stated in BC-2.09.003.

## Proof Harness Skeleton

```go
package config_test

import (
	"testing"
	"time"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/config"
)

func validBaseConfig() config.RouterConfig {
	return config.RouterConfig{
		TickIntervalUpstream:   10 * time.Millisecond,
		TickIntervalDownstream: 50 * time.Millisecond,
		// other required fields omitted; populated by test helper
	}
}

func TestProp_Config_RejectsOutOfRangeTickInterval(t *testing.T) {
	properties := gopter.NewProperties(nil)

	// Below minimum (< 5ms).
	properties.Property("tick_interval < 5ms produces field error", prop.ForAll(
		func(ms uint8) bool {
			if ms == 0 {
				ms = 1
			}
			d := time.Duration(ms%5) * time.Millisecond // 0–4ms
			if d == 0 {
				d = time.Millisecond
			}
			cfg := validBaseConfig()
			cfg.TickIntervalUpstream = d
			errs := config.Validate(cfg)
			if len(errs) == 0 {
				return false
			}
			for _, e := range errs {
				if e.Field == "tick_interval_upstream" || e.Field == "tick_interval_downstream" {
					return true
				}
			}
			return false
		},
		gen.UInt8Range(1, 4),
	))

	// Above maximum (> 50ms).
	properties.Property("tick_interval > 50ms produces field error", prop.ForAll(
		func(extraMs uint8) bool {
			d := 51*time.Millisecond + time.Duration(extraMs)*time.Millisecond
			cfg := validBaseConfig()
			cfg.TickIntervalUpstream = d
			errs := config.Validate(cfg)
			if len(errs) == 0 {
				return false
			}
			for _, e := range errs {
				if e.Field == "tick_interval_upstream" || e.Field == "tick_interval_downstream" {
					return true
				}
			}
			return false
		},
		gen.UInt8(),
	))

	properties.TestingRun(t)
}
```
