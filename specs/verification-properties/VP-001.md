# VP-001: ParseOuterHeader / EncodeOuterHeader Round-Trip Identity

## Property Statement

∀ valid OuterHeader h:
  ParseOuterHeader(EncodeOuterHeader(h)) == h ∧ err == nil

A valid OuterHeader has: Version.Major == 1, SVTN ID is a non-zero 16-byte value,
DstAddr and SrcAddr are 8-byte values, PayloadLen is in [0, 2^32-1].
The round-trip must be exact — no field may be silently truncated, reordered,
or zero-padded differently on decode.

## Source Contract

- **BC:** BC-2.01.004 — Frame Outer-Header Encoding and Decoding at 44-Byte Fixed Layout
- **Postcondition/Invariant:** Postcondition 3 — "The deserialized outer header matches the serialized values exactly (round-trip identity)."

## Proof Method

| Method | Tool | Bounded? | Coverage |

## Proof Harness Skeleton

```go
package frame_test

import (
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/frame"
)

// genValidOuterHeader produces arbitrary valid OuterHeader values.
// All generated headers have Version.Major == 1.
func genValidOuterHeader() gopter.Gen {
	return gopter.CombineGens(
		gen.SliceOfN(16, gen.UInt8()),   // svtn_id
		gen.SliceOfN(8, gen.UInt8()),    // dst_addr
		gen.SliceOfN(8, gen.UInt8()),    // src_addr
		gen.UInt32(),                    // payload_len
		gen.SliceOfN(16, gen.UInt8()),   // hmac (16-byte truncated)
	).Map(func(vals []interface{}) frame.OuterHeader {
		svtnID := vals[0].([]byte)
		dst := vals[1].([]byte)
		src := vals[2].([]byte)
		payloadLen := vals[3].(uint32)
		hmac := vals[4].([]byte)

		h := frame.OuterHeader{}
		h.Version = frame.Version{Major: 1, Minor: 0}
		copy(h.SVTNID[:], svtnID)
		copy(h.DstAddr[:], dst)
		copy(h.SrcAddr[:], src)
		h.PayloadLen = payloadLen
		copy(h.HMAC[:], hmac)
		return h
	})
}

func TestPropRoundTrip_ParseEncodeOuterHeader(t *testing.T) {
	properties := gopter.NewProperties(nil)

	properties.Property("decode(encode(h)) == h for all valid OuterHeaders", prop.ForAll(
		func(h frame.OuterHeader) bool {
			encoded, err := frame.EncodeOuterHeader(h)
			if err != nil {
				return false // encoding of a valid header must not fail
			}
			decoded, err := frame.ParseOuterHeader(encoded)
			if err != nil {
				return false
			}
			return decoded == h
		},
		genValidOuterHeader(),
	))

	properties.TestingRun(t)
}
```
