# VP-018: HalfChannel Emits Empty Frame When No Payload

## Property Statement

∀ HalfChannel H with no queued payload:
  let f = H.Tick()
  len(f.Payload) == 0 ∧ f.FrameType == FrameTypeEmptyTick (0x02)

When `Tick()` is called with no preceding `Enqueue()` call (empty pending queue),
the returned frame's payload field must be zero-length and the frame type must be
`FrameTypeEmptyTick`. The frame is still emitted (exactly one frame per tick —
VP-016) and the sequence counter is still incremented (VP-017).

**Empty-tick case:** Do not call `Enqueue` before `Tick()`. The pending queue is
empty by default. `Enqueue` rejects nil and zero-length payloads with
`ErrEmptyPayload` (BC-2.01.002 precondition 4), so the "nil payload to Tick"
concept from earlier drafts does not apply — the path to an empty tick is simply
calling `Tick()` without any prior `Enqueue()`.

## Source Contract

- **BC:** BC-2.01.001 — Timeslice clock fires on every tick regardless of data availability
- **BC:** BC-2.01.002 — Empty-tick frame is a valid liveness signal
- **Postcondition/Invariant:** Postcondition 2 — "Frame.Payload is zero-length when no data is queued."

## Proof Harness Skeleton

```go
package halfchannel_test

import (
	"testing"
	"time"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/halfchannel"
)

// TestProp_Tick_NilPayload_EmitsEmptyFrame verifies that Tick() with no
// preceding Enqueue call produces a frame with zero-length payload and
// FrameType == FrameTypeEmptyTick.
//
// Empty-tick path: call Tick() without any Enqueue() — the pending queue is
// empty by default. Note that Enqueue(nil) and Enqueue([]byte{}) both return
// ErrEmptyPayload (BC-2.01.002 precondition 4); the "nil payload" concept is
// expressed as "no Enqueue before Tick" in the real API.
func TestProp_Tick_NilPayload_EmitsEmptyFrame(t *testing.T) {
	properties := gopter.NewProperties(nil)

	genDirection := gen.OneConstOf(halfchannel.Upstream, halfchannel.Downstream)

	properties.Property("Tick() with no queued payload produces a zero-length payload frame", prop.ForAll(
		func(chanID uint32, dir halfchannel.Direction) bool {
			hc := halfchannel.New(chanID, dir, 10*time.Millisecond)

			// Empty-tick case: no Enqueue call before Tick().
			f := hc.Tick()
			return len(f.Payload) == 0 && f.FrameType == halfchannel.FrameTypeEmptyTick
		},
		gen.UInt32(),
		genDirection,
	))

	properties.TestingRun(t)
}
```
