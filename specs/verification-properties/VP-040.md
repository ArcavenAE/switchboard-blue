# VP-040: Multipath Failover Recovery < 2s

## Property Statement

When one of two active paths fails (simulated by closing one router connection):
  session traffic recovers on the remaining path within 2 seconds

This covers NFR-003: multipath failover must not require operator
intervention and must complete within 2 seconds of failure detection.

## Source Contract

- **BC:** BC-2.02.003 — Per-path RTT and loss tracked via keep-alive probes; paths ranked by quality
- **NFR-003** (referenced, not a source_bc): failover budget ≤2s
- **Postcondition/Invariant:** When a path's loss crosses red threshold sustained over hysteresis window, multipath stops sending on that path within ≤2s (NFR-003 budget combined with BC-2.02.003 ranking postcondition).

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| e2e | go test (testing.T) | yes | two-path session; kill one path; measure time-to-resume; assert ≤2s |

## Proof Harness Skeleton

```go
//go:build integration

package multipath_test

import (
	"context"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/testenv"
)

// TestE2E_Multipath_FailoverRecovery verifies that closing one of two active
// paths causes traffic to resume on the remaining path within 2 seconds.
func TestE2E_Multipath_FailoverRecovery(t *testing.T) {
	t.Parallel()

	ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
	t.Cleanup(cancel)

	// Two-router topology; session uses both paths.
	env := testenv.NewWithRouters(t, ctx, 2)
	t.Cleanup(env.Close)

	sessionID := env.CreateSession(t)

	// Wait for both paths to be active.
	if err := env.WaitForPaths(t, sessionID, 2, 5*time.Second); err != nil {
		t.Fatalf("did not reach 2 active paths: %v", err)
	}

	// Confirm traffic flowing on both.
	env.SendKeystroke(t, sessionID, "echo pre-fail\n")
	pre := env.CollectFrames(t, sessionID, 2*time.Second)
	if len(pre) == 0 {
		t.Fatal("no frames before path failure")
	}

	// Kill path 0 (close its router connection).
	failAt := time.Now()
	env.CloseRouterConnection(t, 0)

	// Poll until traffic resumes, up to 2s.
	const maxRecovery = 2 * time.Second
	deadline := failAt.Add(maxRecovery)
	var recovered bool
	for time.Now().Before(deadline) {
		env.SendKeystroke(t, sessionID, "echo post-fail\n")
		post := env.CollectFrames(t, sessionID, 200*time.Millisecond)
		if len(post) > 0 {
			recovered = true
			break
		}
	}

	if !recovered {
		elapsed := time.Since(failAt)
		t.Errorf("multipath failover took longer than %v (elapsed %v)", maxRecovery, elapsed)
	}
}
```
