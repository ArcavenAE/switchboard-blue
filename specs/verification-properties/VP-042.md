# VP-042: Keystroke-to-Echo p99 ≤ 100ms

## Property Statement

Over 500 keystroke-to-echo round trips at tick_interval=10ms upstream,
50ms downstream over a single-hop LAN:
  p99(round_trip_latency) ≤ 100ms

This covers NFR-001: the 100ms keystroke-to-echo budget is the primary
interactive latency SLO. Exceedances make the terminal feel unresponsive.

## Source Contract

- **BC:** BC-2.01.001 — HalfChannel Tick Interval (NFR-001)
- **BC:** BC-2.02.001 — Multipath Delivers Within Latency Budget
- **Postcondition/Invariant:** NFR-001 — "Keystroke-to-echo p99 ≤ 100ms at 10ms upstream / 50ms downstream tick."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| benchmark | go test -bench (testing.B) | yes | in-process loopback stack; 500 keystroke/echo round trips; measure wall-clock latency; compute p99 |

## Proof Harness Skeleton

```go
package integration_test

import (
	"context"
	"sort"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/testenv"
)

// BenchmarkKeystrokeToEcho_P99 measures the p99 keystroke-to-echo latency
// over a single-hop loopback stack at the reference tick intervals.
func BenchmarkKeystrokeToEcho_P99(b *testing.B) {
	const (
		upstreamInterval   = 10 * time.Millisecond
		downstreamInterval = 50 * time.Millisecond
		samples            = 500
		maxP99             = 100 * time.Millisecond
	)

	ctx := context.Background()
	env := testenv.NewLoopback(b, ctx, testenv.LoopbackConfig{
		TickIntervalUpstream:   upstreamInterval,
		TickIntervalDownstream: downstreamInterval,
	})
	b.Cleanup(env.Close)

	sessionID := env.CreateSession(b)
	latencies := make([]time.Duration, 0, samples)

	b.ResetTimer()
	for i := 0; i < samples; i++ {
		start := time.Now()
		env.SendKeystroke(b, sessionID, "x")
		env.WaitForEcho(b, sessionID, "x", 500*time.Millisecond)
		latencies = append(latencies, time.Since(start))
	}
	b.StopTimer()

	sort.Slice(latencies, func(i, j int) bool {
		return latencies[i] < latencies[j]
	})

	p99idx := int(float64(len(latencies)) * 0.99)
	p99 := latencies[p99idx]

	b.ReportMetric(float64(p99.Milliseconds()), "p99_rtt_ms")

	if p99 > maxP99 {
		b.Errorf("keystroke-to-echo p99 %v exceeds NFR-001 limit %v", p99, maxP99)
	}
}
```
