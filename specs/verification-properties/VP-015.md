# VP-015: Router Code Never Parses Channel Header Payload

## Property Statement

∀ frame f1, f2 where f1.OuterHeader == f2.OuterHeader ∧ f1.Payload != f2.Payload:
  SVTNRoute(table, f1).OutputInterfaces == SVTNRoute(table, f2).OutputInterfaces
  ∧ SVTNRoute(table, f1).OutputAddresses == SVTNRoute(table, f2).OutputAddresses

The routing decision produced by SVTNRoute must be identical for any two
frames that share the same outer header, regardless of payload content.
Any mutation of payload bytes that changes the routing decision is a
violation of the carrier-grade separation invariant (DI-001).

The code audit belt-and-suspenders verifies that no read of `f.Payload[*]`
or `f.ChannelHeader.*` appears in any call path reachable from SVTNRoute.

## Source Contract

- **BC:** BC-2.01.005 — Channel header is opaque to routers — parseable only by endpoints
- **Postcondition/Invariant:** Postcondition 2 — "The router does not read, log, modify, or cache any bytes at or after offset 44." Invariant DI-001 — "Router cannot read session content."

## Proof Harness Skeleton

```go
package routing_test

import (
	"testing"

	"github.com/arcavenae/switchboard/internal/frame"
	"github.com/arcavenae/switchboard/internal/routing"
)

// FuzzSVTNRoute_PayloadIndependent verifies that mutating payload bytes never
// changes the routing decision. The fuzzer explores the payload space; any
// corpus entry that produces a different routing result than the baseline
// (empty payload, same outer header) is a violation.
func FuzzSVTNRoute_PayloadIndependent(f *testing.F) {
	// Seed: representative frame with known outer header values.
	f.Add([]byte("channel header + payload content that should be ignored by router"))
	f.Add([]byte{})
	f.Add([]byte{0xFF, 0x00, 0xAB, 0xCD, 0xEF})

	f.Fuzz(func(t *testing.T, payload []byte) {
		// Build a routing table with a known SVTN and destination.
		table := routing.NewTable()
		var svtnID [16]byte
		svtnID[0] = 0x01
		var dstAddr [8]byte
		dstAddr[0] = 0xAA
		ifaces := []routing.InterfaceID{routing.InterfaceID(1), routing.InterfaceID(2)}
		table.AddNode(svtnID, dstAddr)
		table.AddRoute(svtnID, dstAddr, ifaces)

		// Baseline frame: same outer header, empty payload.
		baseline := frame.Frame{}
		baseline.OuterHeader.SVTNID = svtnID
		baseline.OuterHeader.DstAddr = dstAddr
		baseline.OuterHeader.PayloadLen = 0
		baseline.Payload = nil

		baselineResult := routing.SVTNRoute(table, baseline, routing.InterfaceID(0))

		// Mutated frame: identical outer header, fuzz-mutated payload.
		mutated := baseline
		mutated.Payload = payload
		mutatedResult := routing.SVTNRoute(table, mutated, routing.InterfaceID(0))

		// Routing decision must be identical regardless of payload.
		if len(baselineResult.OutputInterfaces) != len(mutatedResult.OutputInterfaces) {
			t.Errorf("payload mutation changed output interface count: baseline=%d mutated=%d payload=%x",
				len(baselineResult.OutputInterfaces), len(mutatedResult.OutputInterfaces), payload)
			return
		}
		for i, iface := range baselineResult.OutputInterfaces {
			if i >= len(mutatedResult.OutputInterfaces) || mutatedResult.OutputInterfaces[i] != iface {
				t.Errorf("payload mutation changed output interface[%d]: baseline=%v mutated=%v payload=%x",
					i, iface, mutatedResult.OutputInterfaces[i], payload)
			}
		}
	})
}
```
