# VP-031: tmux Control Mode — 99% Output Event Completeness

## Property Statement

During a sustained 10 KB/s output session via tmux control mode:
  received_events / total_events ≥ 0.99

At least 99% of `%output` events emitted by the tmux process must be
received and parsed by the access node's tmux control mode reader within
the session window. Lost or unparsed events degrade keystroke visibility
and constitute a reliability regression.

## Source Contract

- **BC:** BC-2.04.001 — Access node connects to local tmux via control mode and publishes sessions over SVTN
- **Postcondition/Invariant:** Postcondition — "≥99% of %output events received over a sustained 10KB/s session."

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|

## Proof Harness Skeleton

```go
//go:build integration

package tmux_test

import (
	"context"
	"strings"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/tmux"
)

// TestTmux_ControlMode_OutputCompleteness verifies that the control mode
// reader receives at least 99% of %output events at a 10KB/s generation rate.
func TestTmux_ControlMode_OutputCompleteness(t *testing.T) {
	t.Parallel()

	const (
		duration    = 10 * time.Second
		targetRateB = 10 * 1024 // 10 KB/s
		minPct      = 0.99
	)

	ctx, cancel := context.WithTimeout(context.Background(), duration+5*time.Second)
	t.Cleanup(cancel)

	// Start a tmux session in control mode.
	ctrl, err := tmux.StartControlMode(ctx, tmux.Config{})
	if err != nil {
		t.Fatalf("StartControlMode: %v", err)
	}
	t.Cleanup(func() { _ = ctrl.Close() })

	var received int
	ctrl.OnOutputEvent(func(_ tmux.OutputEvent) {
		received++
	})

	// Generate output inside the session at ~10KB/s.
	emitted := 0
	ticker := time.NewTicker(time.Second)
	t.Cleanup(ticker.Stop)
	deadline := time.Now().Add(duration)
	for time.Now().Before(deadline) {
		// Send a chunk of data to the tmux window; each chunk is one %output event.
		chunk := strings.Repeat("x", targetRateB)
		if err := ctrl.SendToWindow(chunk); err != nil {
			t.Logf("SendToWindow error (non-fatal): %v", err)
		}
		emitted++
		<-ticker.C
	}

	// Allow a brief drain period.
	time.Sleep(500 * time.Millisecond)

	if emitted == 0 {
		t.Fatal("no events emitted; check tmux setup")
	}
	pct := float64(received) / float64(emitted)
	if pct < minPct {
		t.Errorf("output event completeness %.2f%% < required %.0f%% (emitted=%d received=%d)",
			pct*100, minPct*100, emitted, received)
	}
}
```
