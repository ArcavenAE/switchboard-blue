# VP-030: sbctl Exits 1 with E-NET-001 on Connection Refused

## Property Statement

When sbctl targets a non-listening address:
  exit_code == 1
  ∧ stderr contains "E-NET-001"

sbctl must not exit 0, must not hang indefinitely, and must surface the
structured error code E-NET-001 so that scripts and humans can distinguish
a network failure from other error categories.

## Source Contract

- **BC:** BC-2.07.003 — sbctl Surfaces Structured Error Codes on Failure
- **Postcondition/Invariant:** Postcondition — "Connection refused produces exit 1 and E-NET-001 on stderr."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| integration | go test (testing.T) | yes | spawn sbctl binary against a port with no listener; check exit code and stderr |

## Proof Harness Skeleton

```go
//go:build integration

package sbctl_test

import (
	"bytes"
	"os/exec"
	"testing"
)

// TestSbctl_ConnectionRefused_ExitsOneWithENET001 verifies that sbctl
// exits with code 1 and prints E-NET-001 when no daemon is listening.
func TestSbctl_ConnectionRefused_ExitsOneWithENET001(t *testing.T) {
	t.Parallel()

	// Use a port that nothing is listening on.
	target := "127.0.0.1:19999"

	cmd := exec.Command("sbctl", "--target", target, "status")
	var stderr bytes.Buffer
	cmd.Stderr = &stderr

	err := cmd.Run()

	// Must exit non-zero.
	if err == nil {
		t.Fatal("expected sbctl to exit non-zero, but it exited 0")
	}

	exitErr, ok := err.(*exec.ExitError)
	if !ok {
		t.Fatalf("expected *exec.ExitError, got %T: %v", err, err)
	}

	if exitErr.ExitCode() != 1 {
		t.Errorf("expected exit code 1, got %d", exitErr.ExitCode())
	}

	if !bytes.Contains(stderr.Bytes(), []byte("E-NET-001")) {
		t.Errorf("expected stderr to contain E-NET-001; got: %s", stderr.String())
	}
}
```
