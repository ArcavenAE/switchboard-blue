# VP-010: SVTNRoute Never Delivers to Wrong SVTN

## Property Statement

∀ frame f with f.OuterHeader.SVTNID == A,
  admitted_nodes map[SVTNID][]NodeAddr where A ∈ admitted_nodes:
  ∀ dst ∈ SVTNRoute(admitted_nodes, f).OutputAddresses:
    dst ∈ admitted_nodes[A]
  ∧ ∀ B != A: dst ∉ admitted_nodes[B]

No output address produced by SVTNRoute for a frame tagged SVTN-A belongs
to the admitted node set of any other SVTN B. SVTN isolation is enforced
purely by keying the routing table on SVTN ID; no cross-SVTN path exists.

## Source Contract

- **BC:** BC-2.05.006 — SVTN cryptographic isolation: admitted node on SVTN-A cannot see SVTN-B traffic
- **Postcondition/Invariant:** Postcondition 2 — "The router's forwarding engine partitions frame processing by SVTN ID: SVTN-A frames are forwarded only to SVTN-A admitted nodes." Invariant DI-005 — "Cross-SVTN visibility requires possession of keys registered against both SVTNs."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| proptest | gopter v0.2.9+ | no | random routing tables with 2–5 SVTNs and 0–10 nodes each; random frames tagged with one of the SVTNs in the table |

## Proof Harness Skeleton

```go
package routing_test

import (
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/frame"
	"github.com/arcavenae/switchboard/internal/routing"
)

func TestPropSVTNRoute_NeverDeliversToCrossVTN(t *testing.T) {
	properties := gopter.NewProperties(nil)

	properties.Property("SVTNRoute outputs only nodes admitted to the frame's SVTN", prop.ForAll(
		func(svtnCount, nodesPerSVTN int) bool {
			if svtnCount < 2 {
				return true
			}
			svtnCount = (svtnCount % 4) + 2   // 2–5 SVTNs
			nodesPerSVTN = (nodesPerSVTN % 9) + 1 // 1–10 nodes each

			// Build a routing table with svtnCount SVTNs.
			table := routing.NewTable()
			svtnIDs := make([][16]byte, svtnCount)
			for i := range svtnIDs {
				for j := range svtnIDs[i] {
					svtnIDs[i][j] = byte(i*16 + j)
				}
				for n := 0; n < nodesPerSVTN; n++ {
					var addr [8]byte
					addr[0] = byte(i)
					addr[1] = byte(n)
					table.AddNode(svtnIDs[i], addr)
				}
			}

			// Build a frame tagged for svtnIDs[0].
			f := frame.Frame{}
			f.OuterHeader.SVTNID = svtnIDs[0]

			result := routing.SVTNRoute(table, f)

			// All output addresses must be in svtnIDs[0]'s node set.
			svtn0Nodes := table.NodesForSVTN(svtnIDs[0])
			nodeSet := make(map[[8]byte]bool, len(svtn0Nodes))
			for _, n := range svtn0Nodes {
				nodeSet[n] = true
			}

			for _, dst := range result.OutputAddresses {
				if !nodeSet[dst] {
					return false // delivered to wrong SVTN's node
				}
			}
			return true
		},
		gen.IntRange(2, 6),
		gen.IntRange(1, 11),
	))

	properties.TestingRun(t)
}
```
