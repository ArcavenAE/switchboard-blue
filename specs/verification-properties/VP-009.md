# VP-009: Admission Rejects Replayed Nonce

## Property Statement

∀ valid admission (pubkey, nonce N, response) that has been consumed:
  VerifyAdmission(admitted_set, pubkey, N, response) → ErrNonceReplay

Once nonce N has been accepted and consumed from the nonce store, any
subsequent call to VerifyAdmission presenting the same N must return
ErrNonceReplay, regardless of whether the signature is cryptographically
valid. The nonce store is a set; insertion is idempotent but second
verification is a hard reject.

## Source Contract

- **BC:** BC-2.05.001 — Tier 1 SVTN Admission via Signed Key Challenge
- **Postcondition/Invariant:** Postcondition 1 — "The router issues a challenge: a random nonce, signed by the router's own key to prevent replay." The anti-replay guarantee is only meaningful if consumed nonces are tracked and second use is rejected.

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| proptest | gopter v0.2.9+ | no | random admitted key sets with one known key; nonce first admitted successfully, then replayed; verifies second call returns ErrNonceReplay |

## Proof Harness Skeleton

```go
package admission_test

import (
	"crypto/ed25519"
	"crypto/rand"
	"errors"
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/admission"
)

func TestPropNonceReplayRejected(t *testing.T) {
	properties := gopter.NewProperties(nil)

	properties.Property("second VerifyAdmission with same nonce returns ErrNonceReplay", prop.ForAll(
		func(_ int) bool {
			pub, priv, err := ed25519.GenerateKey(rand.Reader)
			if err != nil {
				return true
			}

			admittedSet := admission.NewKeySet()
			admittedSet.Add(pub)

			nonceStore := admission.NewNonceStore()

			var nonce [32]byte
			_, _ = rand.Read(nonce[:])

			resp, err := admission.SignChallenge(priv, nonce)
			if err != nil {
				return true
			}

			// First admission: must succeed.
			ok := admission.VerifyAdmissionWithStore(admittedSet, nonceStore, pub, nonce, resp)
			if !ok {
				return true // first admission failed for unrelated reason; skip
			}

			// Second admission with identical nonce: must return ErrNonceReplay.
			err2 := admission.VerifyAdmissionWithStoreErr(admittedSet, nonceStore, pub, nonce, resp)
			return errors.Is(err2, admission.ErrNonceReplay)
		},
		gen.IntRange(0, 0), // dummy arg; all state is internal
	))

	properties.TestingRun(t)
}
```
