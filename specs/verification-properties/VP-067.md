# VP-067: sbctl Authenticate() Is Fail-Closed — Returns nil Only on Verified AUTH_OK; All Other Outcomes Return Non-nil Error

## Property Statement

The `Authenticate(conn net.Conn, privKey ed25519.PrivateKey) error` function in
`cmd/sbctl/client.go` is **fail-closed**. Specifically:

∀ execution of `Authenticate`:
1. `Authenticate` returns `nil` **if and only if** it successfully received and
   decoded an `{"type":"auth_ok"}` message from the server.

2. `Authenticate` returns a **non-nil error** for every other outcome:
   - (a) connection error on reading CHALLENGE (any `net.Conn` read error)
   - (b) malformed CHALLENGE message (JSON decode error, missing `nonce` field,
     `nonce` not valid base64url-encoded 32 bytes)
   - (c) connection error on sending CHALLENGE_RESPONSE
   - (d) connection error on reading AUTH_OK / AUTH_FAIL response
   - (e) server sends `{"type":"auth_fail"}` — returns error wrapping E-ADM-010
   - (f) server sends any message other than `auth_ok` or `auth_fail`
   - (g) truncated stream (EOF before complete message)
   - (h) oversized AUTH_OK or AUTH_FAIL response (> MaxMessageBytes) —
     `io.LimitReader` triggers error, `Authenticate` returns that error

3. There is **no code path** in `Authenticate` that returns `nil` without
   having decoded a well-formed `{"type":"auth_ok"}` message from the server.

This is the client-side fail-closed invariant that complements the server-side
rejection contract (BC-2.07.004 PC-2/PC-4).

## Source Contract

- **BC:** BC-2.07.002 — sbctl Unified CLI for All Four Daemon Types with OpenSSH Key
  Authentication
- **Postcondition PC-2:** sbctl authenticates the operator's OpenSSH key against the
  daemon's authorized key list.
- **Postcondition PC-4:** if not authenticated: E-ADM-010 "authentication failed";
  exit code 1.
- **ARCH-12 §Authenticate() FAIL-CLOSED Contract:** step 7 — "Return `nil` ONLY if
  an AUTH_OK message was received and successfully decoded."

Note: this VP is anchored on BC-2.07.002 (client-side contract) rather than
BC-2.07.004 (server-side contract) because `Authenticate()` lives in `cmd/sbctl`,
not `internal/mgmt`. The two BCs together specify the full ADR-012 handshake.

## Proof Harness Skeleton

```go
package client_test

import (
    "context"
    "crypto/ed25519"
    "crypto/rand"
    "encoding/json"
    "io"
    "net"
    "testing"

    sbctl "github.com/arcavenae/switchboard/cmd/sbctl"
    "github.com/arcavenae/switchboard/internal/mgmt"
)

// TestAuthenticate_FailClosed_VP067 verifies VP-067:
// Authenticate returns nil only on AUTH_OK; all other outcomes return non-nil error.
func TestAuthenticate_FailClosed_VP067(t *testing.T) {
    t.Parallel()

    opPub, opPriv, err := ed25519.GenerateKey(rand.Reader)
    if err != nil {
        t.Fatalf("generate key: %v", err)
    }
    _ = opPub

    // Helper: start a mock server that sends the given sequence of messages.
    mockServer := func(t *testing.T, responses []map[string]interface{}) net.Conn {
        t.Helper()
        server, client := net.Pipe()
        go func() {
            defer server.Close()
            enc := json.NewEncoder(server)
            for _, msg := range responses {
                if err := enc.Encode(msg); err != nil {
                    return
                }
            }
        }()
        return client
    }

    // Sub-case (e): server sends AUTH_FAIL → Authenticate must return non-nil error.
    t.Run("auth_fail_returns_error", func(t *testing.T) {
        t.Parallel()
        _, daemonPriv, _ := ed25519.GenerateKey(rand.Reader)
        nonce := make([]byte, 32)
        rand.Read(nonce) //nolint:errcheck
        nonceSig := ed25519.Sign(daemonPriv, nonce)

        conn := mockServer(t, []map[string]interface{}{
            {
                "type":       "challenge",
                "nonce":      mgmt.EncodeBase64URL(nonce),
                "daemon_sig": mgmt.EncodeBase64URL(nonceSig),
            },
            {
                "type":    "auth_fail",
                "code":    "E-ADM-010",
                "message": "authentication failed",
            },
        })
        defer conn.Close()

        err := sbctl.Authenticate(conn, opPriv)
        if err == nil {
            t.Error("VP-067 violated: Authenticate returned nil on AUTH_FAIL")
        }
    })

    // Sub-case (g): truncated stream — EOF after CHALLENGE, before response.
    t.Run("truncated_stream_returns_error", func(t *testing.T) {
        t.Parallel()
        _, daemonPriv, _ := ed25519.GenerateKey(rand.Reader)
        nonce := make([]byte, 32)
        rand.Read(nonce) //nolint:errcheck
        nonceSig := ed25519.Sign(daemonPriv, nonce)

        server, client := net.Pipe()
        go func() {
            enc := json.NewEncoder(server)
            enc.Encode(map[string]interface{}{ //nolint:errcheck
                "type":       "challenge",
                "nonce":      mgmt.EncodeBase64URL(nonce),
                "daemon_sig": mgmt.EncodeBase64URL(nonceSig),
            })
            server.Close() // close immediately after CHALLENGE — truncated stream
        }()
        defer client.Close()

        err := sbctl.Authenticate(client, opPriv)
        if err == nil {
            t.Error("VP-067 violated: Authenticate returned nil on truncated stream")
        }
    })

    // Sub-case (b): malformed CHALLENGE (missing nonce field).
    t.Run("malformed_challenge_returns_error", func(t *testing.T) {
        t.Parallel()
        conn := mockServer(t, []map[string]interface{}{
            {"type": "challenge"}, // missing nonce and daemon_sig
        })
        defer conn.Close()

        err := sbctl.Authenticate(conn, opPriv)
        if err == nil {
            t.Error("VP-067 violated: Authenticate returned nil on malformed CHALLENGE")
        }
    })

    // Happy path: AUTH_OK → Authenticate returns nil.
    t.Run("auth_ok_returns_nil", func(t *testing.T) {
        t.Parallel()
        _, daemonPriv, _ := ed25519.GenerateKey(rand.Reader)
        nonce := make([]byte, 32)
        rand.Read(nonce) //nolint:errcheck
        nonceSig := ed25519.Sign(daemonPriv, nonce)

        conn := mockServer(t, []map[string]interface{}{
            {
                "type":       "challenge",
                "nonce":      mgmt.EncodeBase64URL(nonce),
                "daemon_sig": mgmt.EncodeBase64URL(nonceSig),
            },
            {
                "type":           "auth_ok",
                "daemon_version": "0.1.0",
            },
        })
        defer conn.Close()

        err := sbctl.Authenticate(conn, opPriv)
        if err != nil {
            t.Errorf("VP-067: Authenticate returned error on AUTH_OK: %v", err)
        }
    })
}
```
