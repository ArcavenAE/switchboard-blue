# VP-008: Admission Fails for Unregistered Key

## Property Statement

∀ admitted_set AdmittedKeySet, pubkey PublicKey, response ChallengeResponse
    where pubkey ∉ admitted_set:
  VerifyAdmission(admitted_set, pubkey, response) == false

A node presenting a public key that is not registered in the SVTN's
admitted key set must never be admitted, regardless of the cryptographic
validity of the response. This is the enforcement point for fail-closed
posture (BC-2.05.002): if the key is not in the set, the answer is always no.

## Source Contract

- **BC:** BC-2.05.002 — Router rejects non-admitted nodes before forwarding — fail-closed
- **Postcondition/Invariant:** Postcondition 2 — "If the source address is NOT in the admitted set: frame is dropped." Invariant — "Fail-closed: default action for any ambiguous frame is reject, not forward."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| proptest | gopter v0.2.9+ | no | random admitted key sets (0–10 entries) paired with random public keys not in the set; random challenge responses (valid and invalid) |

## Proof Harness Skeleton

```go
package admission_test

import (
	"crypto/ed25519"
	"crypto/rand"
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/admission"
)

func TestPropAdmissionFailsForUnregisteredKey(t *testing.T) {
	properties := gopter.NewProperties(nil)

	properties.Property("VerifyAdmission returns false for any key not in admitted set", prop.ForAll(
		func(setSize int) bool {
			// Build an admitted set of setSize distinct keys.
			admittedSet := admission.NewKeySet()
			for i := 0; i < setSize%10; i++ {
				pub, _, err := ed25519.GenerateKey(rand.Reader)
				if err != nil {
					return true
				}
				admittedSet.Add(pub)
			}

			// Generate a key that is not in the set.
			unregisteredPub, unregisteredPriv, err := ed25519.GenerateKey(rand.Reader)
			if err != nil {
				return true
			}

			// Create a plausible (even cryptographically valid) response.
			var challenge [32]byte
			_, _ = rand.Read(challenge[:])
			resp, err := admission.SignChallenge(unregisteredPriv, challenge)
			if err != nil {
				return true
			}

			result := admission.VerifyAdmission(admittedSet, unregisteredPub, challenge, resp)
			return !result
		},
		gen.IntRange(0, 10),
	))

	properties.TestingRun(t)
}
```
