# VP-019: ARQ.OnAck Never Delivers a Frame Twice

## Property Statement

∀ sequence of (send, ack, deliver) operations on an ARQ instance:
  ∀ sequence number N:
    count(frames delivered with Sequence == N) ≤ 1

No frame with sequence number N may be passed to the deliver callback more
than once, regardless of how many times an ack or retransmission event is
processed for that sequence number.

## Source Contract

- **BC:** BC-2.02.005 — Downstream ARQ with piggybacked ACK and SACK bitmap
- **Postcondition/Invariant:** Invariant — "Delivery is idempotent: acking the same sequence twice must not re-deliver."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| proptest | gopter v0.2.9+ | no | random send/ack/retransmit event sequences; tracks delivered set; asserts no duplicate delivery |

## Proof Harness Skeleton

```go
package arq_test

import (
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/arq"
	"github.com/arcavenae/switchboard/internal/frame"
)

// arqOp encodes a single operation in a generated sequence.
type arqOp struct {
	kind uint8  // 0=send, 1=ack, 2=ack-dup (ack already-acked seq)
	seq  uint16 // sequence number to operate on (bounded to small range)
}

func genARQOps(maxSeq uint16) gopter.Gen {
	return gen.SliceOfN(32, gopter.CombineGens(
		gen.UInt8Range(0, 2),
		gen.UInt16Range(0, maxSeq),
	).Map(func(vals []interface{}) arqOp {
		return arqOp{
			kind: vals[0].(uint8),
			seq:  vals[1].(uint16),
		}
	}))
}

func TestProp_ARQ_NoDuplicateDelivery(t *testing.T) {
	properties := gopter.NewProperties(nil)

	properties.Property("no frame is delivered more than once", prop.ForAll(
		func(ops []arqOp) bool {
			delivered := make(map[uint16]int)
			a := arq.New(arq.Config{WindowSize: 64})
			a.OnDeliver(func(f frame.Frame) {
				delivered[uint16(f.Sequence)]++
			})

			for _, op := range ops {
				switch op.kind {
				case 0: // send
					f := frame.Frame{}
					f.Sequence = uint64(op.seq)
					_ = a.Send(f)
				case 1, 2: // ack (including duplicate ack)
					a.OnAck(uint64(op.seq))
				}
			}

			for seq, count := range delivered {
				if count > 1 {
					_ = seq // suppress unused warning in skeleton
					return false
				}
			}
			return true
		},
		genARQOps(15),
	))

	properties.TestingRun(t)
}
```
