# VP-016: HalfChannel.Tick Emits Exactly One Frame Per Tick

## Property Statement

∀ (state HalfChannelState, payload []byte):
  let frames = Tick(state, payload)
  len(frames) == 1

Each call to `Tick` must return exactly one `Frame`, regardless of payload
content or current channel state. Callers depend on this invariant to pace
outbound traffic at a fixed rate without buffering or back-pressure logic.

## Source Contract

- **BC:** BC-2.01.001 — Timeslice clock fires on every tick regardless of data availability
- **BC:** BC-2.01.003 — Upstream and downstream half-channels operate with independent clocks and sequence spaces
- **Postcondition/Invariant:** Postcondition 1 — "Tick returns exactly one Frame."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| proptest | gopter v0.2.9+ | no | random sampling over (HalfChannelState, []byte) pairs; asserts exactly one frame returned |

## Proof Harness Skeleton

```go
package halfchannel_test

import (
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/halfchannel"
)

func genHalfChannelState() gopter.Gen {
	return gopter.CombineGens(
		gen.UInt64(), // sequence counter
		gen.UInt8(),  // flags
	).Map(func(vals []interface{}) halfchannel.State {
		return halfchannel.State{
			Sequence: vals[0].(uint64),
			Flags:    vals[1].(uint8),
		}
	})
}

func TestProp_Tick_EmitsExactlyOneFrame(t *testing.T) {
	properties := gopter.NewProperties(nil)

	properties.Property("Tick returns exactly one frame for any (state, payload)", prop.ForAll(
		func(state halfchannel.State, payload []byte) bool {
			ch := halfchannel.New(state)
			frames := ch.Tick(payload)
			return len(frames) == 1
		},
		genHalfChannelState(),
		gen.SliceOf(gen.UInt8()),
	))

	properties.TestingRun(t)
}
```
