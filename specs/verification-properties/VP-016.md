# VP-016: HalfChannel.Tick Emits Exactly One Frame Per Tick

## Property Statement

∀ HalfChannel H, ∀ call to H.Tick():
  exactly one ChannelFrame is returned.

Each call to `Tick()` must return exactly one `ChannelFrame`, regardless of
payload content or current channel state. Callers depend on this invariant to
pace outbound traffic at a fixed rate without buffering or back-pressure logic.

**Structural enforcement note:** The Go type signature `func (h *HalfChannel) Tick() ChannelFrame`
makes this property structurally enforced at compile time — the return type is a
single value, not a slice or pointer. There is no runtime gopter test in S-1.02
(see `halfchannel_test.go` lines 14–23 top-of-file comment) because the type system
handles the "exactly one" cardinality guarantee. VP-016's gopter harness skeleton is
provided as a Phase-6 proptest reference to exercise accompanying properties (ChanID
propagation, FrameType selection) under randomised input, not to assert frame count
at runtime.

## Source Contract

- **BC:** BC-2.01.001 — Timeslice clock fires on every tick regardless of data availability
- **BC:** BC-2.01.003 — Upstream and downstream half-channels operate with independent clocks and sequence spaces
- **Postcondition/Invariant:** Postcondition 1 — "Tick returns exactly one Frame."

## Proof Harness Skeleton

```go
package halfchannel_test

import (
	"testing"
	"time"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/halfchannel"
)

// TestProp_Tick_EmitsExactlyOneFrame verifies that Tick() always produces
// exactly one ChannelFrame. The "exactly one" cardinality is structurally
// enforced by the return type — this proptest additionally exercises ChanID
// propagation and FrameType selection under randomised input.
func TestProp_Tick_EmitsExactlyOneFrame(t *testing.T) {
	properties := gopter.NewProperties(nil)

	// Generate a random chanID and direction.
	genDirection := gen.OneConstOf(halfchannel.Upstream, halfchannel.Downstream)

	properties.Property("Tick() returns a frame carrying the configured ChanID", prop.ForAll(
		func(chanID uint32, dir halfchannel.Direction) bool {
			hc := halfchannel.New(chanID, dir, 10*time.Millisecond)
			f := hc.Tick()
			// Primary invariant: ChanID is propagated to the frame.
			if f.ChanID != chanID {
				return false
			}
			// Empty-tick case: no Enqueue call, so FrameType must be EMPTY_TICK.
			return f.FrameType == halfchannel.FrameTypeEmptyTick
		},
		gen.UInt32(),
		genDirection,
	))

	properties.Property("Tick() after Enqueue() returns a frame with FrameTypeData", prop.ForAll(
		func(chanID uint32, payload []byte) bool {
			if len(payload) == 0 {
				return true // skip: Enqueue rejects empty payload (ErrEmptyPayload)
			}
			hc := halfchannel.New(chanID, halfchannel.Upstream, 10*time.Millisecond)
			if err := hc.Enqueue(payload); err != nil {
				return false
			}
			f := hc.Tick()
			return f.ChanID == chanID &&
				f.FrameType == halfchannel.FrameTypeData &&
				string(f.Payload) == string(payload)
		},
		gen.UInt32(),
		gen.SliceOf(gen.UInt8()),
	))

	properties.TestingRun(t)
}
```
