# VP-014: DeriveNodeAddress Is Deterministic

## Property Statement

∀ svtn_id [16]byte, pubkey ed25519.PublicKey:
  DeriveNodeAddress(svtn_id, pubkey) == DeriveNodeAddress(svtn_id, pubkey)

The function is pure: same (svtn_id, pubkey) always produces the identical
8-byte node address. No randomness, timestamps, or global state may
influence the output. This property also implies:

  DeriveNodeAddress(svtn_id1, pubkey) != DeriveNodeAddress(svtn_id2, pubkey)
    for svtn_id1 != svtn_id2

(same key on different SVTNs yields different addresses), which is verified
as a secondary sub-property in the harness below.

## Source Contract

- **BC:** BC-2.01.006 — Session identity is cryptographic: node address derived from hash(SVTN-ID, public-key)
- **Postcondition/Invariant:** Postcondition 2 — "The same (SVTN-ID, public-key) pair always produces the same address (deterministic)." Postcondition 3 — "Different SVTNs produce different addresses for the same keypair."

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| proptest | gopter v0.2.9+ | no | random (svtn_id, pubkey) pairs; each pair called twice; result equality checked; secondary test uses two distinct svtn_ids and same pubkey |

## Proof Harness Skeleton

```go
package frame_test

import (
	"bytes"
	"crypto/ed25519"
	"crypto/rand"
	"testing"

	"github.com/leanovate/gopter"
	"github.com/leanovate/gopter/gen"
	"github.com/leanovate/gopter/prop"

	"github.com/arcavenae/switchboard/internal/frame"
)

func TestPropDeriveNodeAddress_Deterministic(t *testing.T) {
	properties := gopter.NewProperties(nil)

	// Primary: same inputs → same output.
	properties.Property("DeriveNodeAddress is deterministic: same (svtn_id, pubkey) → same address", prop.ForAll(
		func(svtnIDBytes []byte) bool {
			pub, _, err := ed25519.GenerateKey(rand.Reader)
			if err != nil {
				return true
			}

			var svtnID [16]byte
			copy(svtnID[:], svtnIDBytes)

			addr1 := frame.DeriveNodeAddress(svtnID, pub)
			addr2 := frame.DeriveNodeAddress(svtnID, pub)
			return addr1 == addr2
		},
		gen.SliceOfN(16, gen.UInt8()),
	))

	// Secondary: different SVTN IDs → different addresses for same key.
	properties.Property("DeriveNodeAddress produces different addresses for different SVTN IDs", prop.ForAll(
		func(id1Bytes, id2Bytes []byte) bool {
			var id1, id2 [16]byte
			copy(id1[:], id1Bytes)
			copy(id2[:], id2Bytes)

			if bytes.Equal(id1[:], id2[:]) {
				return true // skip equal IDs; not the case we're testing
			}

			pub, _, err := ed25519.GenerateKey(rand.Reader)
			if err != nil {
				return true
			}

			addr1 := frame.DeriveNodeAddress(id1, pub)
			addr2 := frame.DeriveNodeAddress(id2, pub)
			return addr1 != addr2
		},
		gen.SliceOfN(16, gen.UInt8()),
		gen.SliceOfN(16, gen.UInt8()),
	))

	properties.TestingRun(t)
}
```
