# VP-076: Bootstrap Key Non-Revocable AND Non-Expirable Invariant (Symmetric Management-Lockout Prevention)

## Property Statement

∀ `admin.key.revoke` or `admin.key.expire` RPC where the target key is the bootstrap
key (i.e., the public key passed at SVTN creation — `m.controlPubKey`):

1. **Revoke path:** `SVTNManager.RevokeKey` MUST return `svtnmgmt.ErrBootstrapKeyRevokeForbidden`
   unconditionally, regardless of whether other control keys have been registered.
   The `admin.key.revoke` handler MUST map this sentinel to E-ADM-020 in-band response
   (`ok: false`); the key store MUST NOT be mutated.

2. **Expire path:** `SVTNManager.ExpireKey` MUST return `svtnmgmt.ErrBootstrapKeyExpireForbidden`
   unconditionally, regardless of whether other control keys have been registered.
   The `admin.key.expire` handler MUST map this sentinel to E-ADM-021 in-band response
   (`ok: false`); the key store MUST NOT be mutated.

3. **Symmetry invariant:** The two forbidden-sentinel guards MUST be structurally identical
   in `svtnmgmt.go` — both BEFORE the SVTN lookup, both using a constant-time compare
   against `m.controlPubKey`. A bootstrap-key target MUST never reach the SVTN lookup
   step for either revoke or expire.

4. The connection MUST remain open after either rejection (both are in-band `ok: false`
   responses, not connection-close events).

The property directly enforces BC-2.05.004 EC-007 v1.10: "the bootstrap key cannot be
revoked or expired at any time." Per BC-2.05.004 EC-004, expiry has the same functional
lockout effect as revocation; therefore the protection MUST be symmetric.

## Source Contract

- **BC:** BC-2.05.004 — Key Lifecycle — Register, Revoke, and Expire Admission and
  Session-Authorization Keys
- **EC-007 (v1.10):** "Operator attempts to revoke OR expire the bootstrap key (permanent
  trust anchor). Both revoke and expire return their respective sentinel unconditionally
  — the bootstrap key cannot be revoked or expired at any time, regardless of whether
  other control keys have been registered. ErrBootstrapKeyRevokeForbidden → E-ADM-020
  for revoke; ErrBootstrapKeyExpireForbidden → E-ADM-021 for expire."
- **EC-004:** "Key expires while session is active — same behavior as revocation" (establishes
  that expiry = functional lockout, motivating symmetric protection).
- **Error Taxonomy:** E-ADM-020 (bootstrap-key-revoke-forbidden) and E-ADM-021
  (bootstrap-key-expire-forbidden) per error-taxonomy.md v3.8.
- **Capability Anchor:** CAP-019 ("Key lifecycle management (register, revoke, expire)")
  per capabilities.md §CAP-019.

## Proof Harness Skeleton

```go
//go:build integration

package main

import (
	"context"
	"crypto/ed25519"
	"crypto/rand"
	"net"
	"path/filepath"
	"testing"
	"time"

	"github.com/arcavenae/switchboard/internal/admission"
	"github.com/arcavenae/switchboard/internal/mgmt"
	"github.com/arcavenae/switchboard/internal/svtnmgmt"
)

// TestIntegration_BootstrapKey_RevokeAndExpireForbidden_VP076 verifies VP-076:
// both revoke and expire of the bootstrap key return their respective forbidden
// sentinel unconditionally (E-ADM-020 / E-ADM-021), with connection kept open.
//
// Helper functions authenticateAdminClient / sendAdminRPC / readAdminRPCResponse
// follow the canonical pattern in cmd/switchboard/admin_handlers_e2e_test.go.
func TestIntegration_BootstrapKey_RevokeAndExpireForbidden_VP076(t *testing.T) {
	t.Parallel()

	// Generate the bootstrap (control) keypair — this is m.controlPubKey.
	bootstrapPub, bootstrapPriv, err := ed25519.GenerateKey(rand.Reader)
	if err != nil {
		t.Fatalf("generate bootstrap key: %v", err)
	}

	// Build SVTNManager and register the bootstrap key as a control key.
	svtnMgr := svtnmgmt.New()
	if _, err := svtnMgr.Create("test-svtn"); err != nil {
		t.Fatalf("create test-svtn: %v", err)
	}
	if _, err := svtnMgr.RegisterKey("test-svtn", bootstrapPub, admission.RoleControl); err != nil {
		t.Fatalf("register bootstrap key: %v", err)
	}

	// Include bootstrap key in operator set so auth passes.
	ops := mgmt.NewOperatorKeySet([]ed25519.PublicKey{bootstrapPub})

	// Build admin handlers via the S-6.06 AC-001 contract entry point.
	handlers := BuildAdminHandlers(svtnMgr, ops)

	subCases := []struct {
		name        string
		cmd         string
		args        map[string]interface{}
		expectedErr string
	}{
		{
			name: "bootstrap_revoke_forbidden",
			cmd:  "admin.key.revoke",
			args: map[string]interface{}{
				"svtn_id": "test-svtn",
				"key":     "ssh-ed25519 AAAA...", // bootstrap pubkey wire form
				"role":    "control",
			},
			expectedErr: "E-ADM-020",
		},
		{
			name: "bootstrap_expire_forbidden",
			cmd:  "admin.key.expire",
			args: map[string]interface{}{
				"svtn_id": "test-svtn",
				"key":     "ssh-ed25519 AAAA...", // bootstrap pubkey wire form
				"after":   "720h",
			},
			expectedErr: "E-ADM-021",
		},
	}

	for _, tc := range subCases {
		tc := tc
		t.Run(tc.name, func(t *testing.T) {
			t.Parallel()

			sockDir := t.TempDir()
			sockPath := filepath.Join(sockDir, "mgmt.sock")
			ln, err := net.Listen("unix", sockPath)
			if err != nil {
				t.Fatalf("listen: %v", err)
			}
			t.Cleanup(func() { ln.Close() })

			_, daemonPriv, err := ed25519.GenerateKey(rand.Reader)
			if err != nil {
				t.Fatalf("generate daemon key: %v", err)
			}

			ctx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
			t.Cleanup(cancel)
			go func() {
				_ = mgmt.NewServer(ln, daemonPriv, ops, handlers, "dev").Serve(ctx)
			}()

			conn, err := net.Dial("unix", sockPath)
			if err != nil {
				t.Fatalf("dial: %v", err)
			}
			t.Cleanup(func() { conn.Close() })

			// Authenticate as the bootstrap key (control-role, auth passes).
			authenticateAdminClient(t, conn, bootstrapPriv, daemonPriv.Public().(ed25519.PublicKey))

			// Issue the forbidden operation targeting the bootstrap pubkey.
			sendAdminRPC(t, conn, tc.cmd, tc.args)

			// Expect in-band ok:false with the forbidden-sentinel error code.
			resp := readAdminRPCResponse(t, conn)
			if resp["ok"] != false {
				t.Errorf("expected ok:false, got ok:%v", resp["ok"])
			}
			code, _ := resp["code"].(string)
			if code != tc.expectedErr {
				t.Errorf("expected %s, got code:%q", tc.expectedErr, code)
			}

			// Verify connection is still open (second RPC succeeds).
			sendAdminRPC(t, conn, "admin.key.list-keys", map[string]interface{}{
				"svtn_id": "test-svtn",
			})
			resp2 := readAdminRPCResponse(t, conn)
			if resp2["ok"] != true {
				t.Errorf("connection must remain open after forbidden sentinel; list-keys got ok:%v", resp2["ok"])
			}
		})
	}
}
```
