# S-7.03: Console Remote Control via sbctl

> **Execute:** `/vsdd-factory:deliver-story S-7.03`

## Narrative

- **As an** operator
- **I want to** remotely control a console daemon via sbctl (attach, detach, switch session, navigate)
- **So that** I can manage console sessions from a management station without interactive console access

## Acceptance Criteria

### AC-001 (traces to BC-2.08.001 postcondition 1)
`sbctl console attach --target <console_addr> --session <name>` causes the console daemon to attach to the named session.
- **Test:** `TestSbctlConsole_Attach`

### AC-002 (traces to BC-2.08.001 postcondition 2)
`sbctl console detach --target <console_addr>` causes the console daemon to detach from the current session without closing it.
- **Test:** `TestSbctlConsole_Detach`

### AC-003 (traces to BC-2.08.001 postcondition 3)
`sbctl console switch --target <console_addr> --session <name>` detaches from the current session and attaches to the named session in one atomic operation.
- **Test:** `TestSbctlConsole_Switch`
